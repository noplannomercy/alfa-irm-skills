## 미팅 중 감지할 신호
- "센터마다 절차가 달라서..." → GraphRAG 필요 신호
- "고객한테 직접 챗봇으로..." → 2단계 접근 필요
- "클라우드 안 되고..." → 온프레미스 분기
- "전에 챗봇 넣었는데..." → PoC 실패 이력
- "대표님이 빨리..." → 일정 압박

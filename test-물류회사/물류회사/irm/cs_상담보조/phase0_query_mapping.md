## 체이닝 패턴 (복합 질의)

- **"반품 처리해줘"**
  = 탐색(반품 절차 조회) → 접근(주문 정보+배송 상태) → 실행(확정, 가능 판단)

- **"이 클레임 어떻게 처리해?"**
  = 접근(상담 이력) → 탐색(클레임 가이드) → 실행(유연, 보상 기준 적용 제안)

- **"배송 지연 안내 보내야 해"**
  = 접근(배송 현황) → 탐색(안내 문구) → 실행(유연, 초안 생성)

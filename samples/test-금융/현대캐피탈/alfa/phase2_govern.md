## 보조수단성
- AI는 정보 제공(보조)만. 심사 결정 참여 아님.
- 심사역이 AI 결과를 무시하고 다른 결정 가능해야 함.
- AI 추천 vs 심사역 결정 차이 로그 필수.
- 심사 의견서에 "AI 보조 활용" + 참조 규정 근거 명시.

## 신용정보 차단
- 신용정보법 제32조 — LLM에 신용정보 절대 투입 불가
- 규정/내규/약관/가이드라인 = OK (신용정보 아님)
- CBS 신용정보 → LLM 전달 경로 구조적 차단 필수

## 감사/기록
- 감사로그: 모든 AI 호출 입력/출력/타임스탬프/사용자 기록
- AI 감사: 파일럿 분기 1회, 운영 연 1회
- 모델 정책: 사용 모델별 데이터 처리 정책 문서 제출

## 보안
- 데이터 암호화: TLS(전송) + AES-256(저장)
- Bedrock: 데이터 외부 미전송, 모델 학습 미사용 (AWS 정책)

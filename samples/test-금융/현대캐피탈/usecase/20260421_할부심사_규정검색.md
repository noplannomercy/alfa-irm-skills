# 유즈케이스 정의서: 자동차 할부 심사 규정 검색 + 의견서 보조

> 고객: 현대캐피탈
> 작성일: 2026-04-21
> 상태: 확정

## 개요
- 목적: 심사역이 규정집 2,000페이지에서 해당 규정을 빠르게 찾고, 심사 의견서 초안을 보조받는 내부 도구
- 사용자: 본사 심사팀 심사역 10명 (파일럿)
- 입력: 심사 건 정보 (상품유형, 심사 항목, 질문)
- 출력: 관련 규정 검색 결과 + 심사 의견서 초안 (참조 근거 표시)

## 행위 유형 분포 (예상)

| 행위 유형 | 비중 | 설명 |
|----------|------|------|
| 탐색(내부) | 60% | 규정집 검색, 가이드라인 검색, 심사 사례 검색 |
| 접근 | 20% | CBS 심사 건 조회 (비신용정보 항목만) |
| 실행(유연) | 20% | 심사 의견서 초안 생성 (LLM 판단) |
| 실행(확정) | 0% | 해당 없음 |

## 리소스 현황

| # | 리소스명 | 유형 | 근거 | 위치/접근 방법 | 규모 |
|---|---------|------|:----:|-------------|------|
| 1 | 여신규정 | 문서/파일 | ✅ | PDF (PoC 잔여본 + 원본 수령 예정) | ~800p |
| 2 | 자동차금융 특약 | 문서/파일 | ✅ | PDF | ~600p |
| 3 | 리스 특약 | 문서/파일 | ⚠️ | PDF (이번 파일럿 범위 밖이나 GraphRAG 필요) | ~400p |
| 4 | 심사 가이드라인 | 문서/파일 | ✅ | PDF/HWP | ~200p (규정 해석 + 사례집) |
| 5 | CBS 심사 건 정보 | 시스템 | ✅ | Oracle DB, REST API (신용정보 제외 항목만) | — |
| 6 | 심사 이력 (비식별화) | 시스템 | ⚠️ | CBS → 비식별화 후 (의미 있는지 미확인) | — |

**리소스 신뢰도: ✅4건 (67%) / ⚠️2건 (33%)**
- 리스 특약: 파일럿 범위 밖이나 GraphRAG 상품-규정 관계에 필요
- 심사 이력: 비식별화 후 활용 가치 미확인 (컴플라이언스 우려)

## GraphRAG 필요성
- **필요 (강한 신호 — PoC 실패 원인)**
- 근거:
  - 같은 "담보 평가" 조항이 여신규정/자동차특약/리스특약에 각각 다르게 존재
  - 상품(할부/리스) + 규정유형(여신/특약) 2차원 분기
  - PoC 실패 원인 2번 = 이 구분을 못 해서 실패
  - 규정 계층(편→장→조→항) + 상품 분기 = 트리 구조 → 그래프 탐색 적합

# 현대캐피탈 1차 미팅 메모

> 일시: 2026-04-14
> 참석: 이준혁 디지털혁신팀 팀장, 송미래 디지털혁신팀 과장, 최동원 심사기획팀 차장, 우리 측 2명

## 미팅 내용 (원문)

- 디지털혁신팀 이준혁 팀장이 프로젝트 리드. 팀 8명, AI/데이터 담당.
- 심사기획팀 최동원 차장이 현업 대표로 동석.

- 심사 업무 현황:
  - 자동차 할부/리스 심사가 메인. 개인대출도 있지만 이번은 자동차 심사만.
  - 심사역 약 80명 (본사 + 지점). 하루 심사 건수 약 1,500건.
  - 심사 프로세스: 딜러가 고객 서류 접수 → 자동 스코어링(CSS) → 심사역이 서류 확인 + 추가 심사 → 승인/거절/조건부 승인
  - 자동 스코어링으로 70%는 자동 처리. 나머지 30%(하루 450건)를 심사역이 수동 심사.
  - 수동 심사 시 확인하는 것: 소득 증빙, 재직 증명, 기존 대출 현황, 차량 담보 가치, 내규 적합성
  - 심사역 최동원 차장: "규정집이 2,000페이지고, 내규가 수시로 바뀌는데 그걸 다 외우고 있을 수 없다. 새 심사역은 6개월은 배워야 혼자 심사 가능."
  - "심사 의견서 쓰는 데도 시간이 걸린다. 건당 20~30분. 하루에 10건 이상 쓰면 녹초."

- PoC 실패 상세:
  - 1년 전 오토에버가 진행. "심사 자동화 PoC"라고 했음.
  - 시도한 것: 규정집을 RAG로 넣고, 심사 질문에 답하게 하려 함.
  - 실패 원인 (이준혁 팀장 판단):
    1) "규정집 2,000페이지를 통째로 넣었더니 관련 없는 규정이 튀어나왔다"
    2) "같은 질문인데 자동차 할부랑 리스랑 규정이 다른 걸 구분 못 했다"
    3) "심사역들한테 '이거 써라'고 했더니 '내가 왜 기계한테 물어봐야 하냐'고 반발"
    4) "금감원 보고 어떻게 할 건지 아무도 답을 못 했다"
  - 남은 자산: 규정집 PDF 전환본, AWS 계정 (Bedrock 활성화 안 됨)

- 오토에버 관계:
  - 현대차그룹 IT 계열사라 완전 배제는 어려움
  - 이번 6개월 파일럿은 오토에버가 PM + 인프라 담당
  - "우리는 AI/LLM 전문성이 필요한 부분을 맡아달라"고 함
  - 역할: 오토에버 = PM + 인프라 + SI, 우리 = AI 설계 + 구현 + 자문
  - 이준혁 팀장: "솔직히 오토에버는 AI를 모른다. 근데 그룹사라 빼기도 그렇고. 실질적으로는 여러분이 AI 파트 전체를 리드해줘야 한다."

- 인프라:
  - AWS 사용 중. EC2, RDS(Oracle), S3. Bedrock은 아직 활성화 안 함.
  - 심사 시스템(CBS)은 Oracle DB. API는 내부용 있음 (REST).
  - 데이터 레이크: 없음. S3에 로그 쌓이는 정도.
  - SageMaker: 계정만 있고 안 쓰고 있음.
  - 이준혁 팀장: "Bedrock 쓸 수 있으면 쓰고 싶은데, 신용정보 넣어도 되는 건지 모르겠다."

- 규제:
  - 컴플라이언스팀 있음. 팀장급. 이번 프로젝트 인지하고 있음.
  - "2차 미팅에 컴플라이언스 동석 가능. 근데 그분이 까다로워서 미리 정리해서 가야 한다."
  - 금감원: AI 활용 시 "보조수단성" 확보 필수. 최종 심사 결정은 반드시 사람(심사역).
  - 신용정보: CBS에 있는 신용정보(신용점수, 연체이력, 소득)를 LLM에 직접 넣는 건 안 됨. 
  - 이준혁 팀장: "신용정보는 못 넣는다는 건 알겠는데, 그러면 AI가 뭘 할 수 있는 거냐가 핵심 질문이다."

- 원하는 것 (좁히기):
  - 최동원 차장: "나는 규정 검색이 가장 급하다. 2,000페이지에서 해당 규정 찾는 게 반나절이다."
  - 이준혁 팀장: "심사 의견서 초안 자동 생성까지 되면 좋겠다."
  - 합의: "우선 '규정 검색 + 심사 의견서 초안 보조'로 좁히자. 신용정보 직접 투입은 빼고."
  - 파일럿 범위: 자동차 할부 심사 → 규정 검색 + 심사 의견서 보조

- 예산: 6개월 파일럿 전체 7억 (오토에버 + 우리 포함). 우리 몫은 2~3억 규모.
- 일정: 5월 착수 → 10월 종료. 11월 경영진 보고.
- 인력: 디지털혁신팀 송미래 과장이 실무 PM. 오토에버 5명 + 우리 2~3명.

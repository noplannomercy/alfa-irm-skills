## 행위 유형별 검증

### 탐색 검증
| 항목 | 기준 | 측정 방법 |
|------|------|----------|
| 규정 검색 정확도 | 해당 상품의 정확한 조항 top-5 중 3건 적중 (60%) | 테스트 질의 30건, 심사역 판정 |
| **상품 구분 정확도** | 할부 질문에 리스 규정 반환 0건 | 상품 교차 질의 20건 — **PoC 실패 재현 테스트** |
| GraphRAG 경로 | Topic→Article→Product 정확 추적 | 경로 20건 수동 검증 |
| 가이드라인 사례 적중 | 유사 사례 top-3 중 1건 이상 관련 | 테스트 15건 |
| 응답 시간 | 5초 이내 | 평균 측정 |

### 접근 검증
| 항목 | 기준 | 측정 방법 |
|------|------|----------|
| **신용정보 차단** | 프록시 통과 0건 — **절대 기준** | 샘플 50건, 전 필드 검증 |
| 감사로그 | 모든 조회 기록 100% | 20건 후 로그 대조 |

### 실행 검증
| 항목 | 기준 | 측정 방법 |
|------|------|----------|
| 의견서 초안 채택률 | 심사역 70% 수용 (수정 없이 또는 경미) | 파일럿 100건 |
| 근거 표시 정확도 | 100% (모든 의견서에 조항 번호 포함) | 전수 확인 |
| 보조수단성 | 심사역 override 기능 정상 | 의도적 override 10건 테스트 |
| 차이 로그 | override 시 자동 기록 100% | 전수 확인 |

## PoC 범위
- 대상: 본사 심사팀 심사역 10명
- 기간: 4주 운영
- 성공 기준:
  - 규정 검색 시간 50% 단축 (현재 반나절 → 목표 2~3시간 이내)
  - 상품 구분 오류 0건 (PoC 실패 재현 방지)
  - 신용정보 노출 0건
  - 심사역 "쓸만하다" 정성 평가
  - 의견서 초안 채택률 70%
- 확대 조건: 성공 시 전체 심사역 80명 + 리스 심사 확대

### 답변 초안 생성
- [ ] 체이닝 파이프라인 구현 (5단계)
  - [ ] Step 1: 문의 내용 입력
  - [ ] Step 2: knowledge_base 검색 (FAQ+매뉴얼) — 병렬
  - [ ] Step 3: consultation_history 검색 (유사 이력) — 병렬
  - [ ] Step 4: cli-crm 고객 이력 조회 (선택, 병렬) + 개인정보 마스킹
  - [ ] Step 5: LLM 답변 초안 생성
- [ ] LLM 프롬프트 설계
  - [ ] 입력: 문의 + FAQ 결과 + 매뉴얼 결과 + 이력 결과
  - [ ] 출력: 답변 초안 (고객 응대 톤)
  - [ ] 톤 가이드: 기존 상담 톤 참조 (CS팀과 합의)
- [ ] 개인정보 필터링

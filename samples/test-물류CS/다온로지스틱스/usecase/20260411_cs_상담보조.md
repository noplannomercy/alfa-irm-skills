# 유즈케이스 정의서: CS 상담 보조

> 고객: 다온로지스틱스
> 작성일: 2026-04-11
> 상태: 확정

## 개요
- 목적: 상담사가 반복 질문 응대 시 FAQ/매뉴얼을 빠르게 찾고 답변 초안을 받는 내부 보조 도구
- 사용자: CS팀 상담사 15명 (파일럿 5명)
- 입력: 상담사가 입력한 질문 또는 고객 문의 내용
- 출력: 관련 FAQ/매뉴얼 검색 결과 + 답변 초안

## 행위 유형 분포 (예상)

| 행위 유형 | 비중 | 설명 |
|----------|------|------|
| 탐색(내부) | 70% | FAQ 검색, 매뉴얼 검색, 유사 상담 이력 검색 |
| 접근 | 15% | CRM 상담 이력 조회 (특정 건 조회) |
| 실행(유연) | 15% | 답변 초안 생성 (LLM 판단) |
| 실행(확정) | 0% | 해당 없음 |

## 리소스 현황

| # | 리소스명 | 유형 | 근거 | 위치/접근 방법 | 규모 |
|---|---------|------|:----:|-------------|------|
| 1 | FAQ | 문서/파일 | ✅ | 구글 시트 | 300건 |
| 2 | 상담 매뉴얼 | 문서/파일 | ✅ | Notion | 50건 |
| 3 | 상담 이력 | 시스템(CRM) | ✅ | PostgreSQL, REST API | 2년 50만건 |

**리소스 신뢰도: ✅3건 (100%)**

# 다온로지스틱스 1차 미팅 서머리

> 일시: 2026-04-11
> 참석: 한지영 CS팀장, 오민수 IT담당, 당사 2명

## 논의 내용

1. **CS팀 현황**: 상담사 15명, 일 800건 처리. 채널은 카카오톡(60%), 전화(30%), 이메일(10%). 반복 질문(배송 추적/반품/집하 예약)이 전체의 60% 차지.
2. **대상 업무 확정**: 상담사용 내부 보조 도구 — 질문 입력 시 FAQ/매뉴얼에서 관련 내용 검색 + 답변 초안 제시.
3. **데이터 현황**: FAQ 300건(구글 시트), 상담 매뉴얼 50건(Notion), 상담 이력 2년치 50만건(CRM).
4. **인프라**: AWS 기반(EC2, RDS, S3), Bedrock 활용 가능.
5. **이전 경험**: 1년 전 규칙 기반 챗봇 도입 후 3개월 만에 철거. 이번에는 상담사가 직접 활용하는 보조 도구 형태로 접근.
6. **확장 계획**: CS팀에서 효과 확인 후 물류센터 매뉴얼 검색, 셀러 문의 대응 등으로 확대 검토.

## 합의 사항
- 파일럿 대상: CS팀 상담 보조 (내부 도구)
- 고객 직접 대면 아님 (내부 상담사용)
- 예산 2천만원, 6주 내 파일럿 결과 보고
- IT담당 오민수님 풀타임 투입

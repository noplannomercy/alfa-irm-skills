## A. 사전 질의 (CS팀장님용)

### 현황 관련
- 현재 상담 채널이 어떻게 되시나요? (전화, 카카오톡, 웹챗, 이메일 등)
- 하루 평균 상담 건수와 상담사 1인당 처리량이 어느 정도인가요?
- 가장 많이 들어오는 반복 질문 유형 top 3~5개를 알려주세요
- FAQ 문서가 현재 몇 건 정도 있고, 최근 업데이트는 언제 했는지 알려주세요
- 이전에 챗봇이나 자동 응답 시스템을 도입해본 적이 있으시면 경험을 공유해주세요

### 목표/기대 관련
- AI 도입으로 기대하시는 구체적인 효과가 있으시면 알려주세요 (응답 시간 단축, 상담사 업무 경감 등)
- 언제까지 결과를 보고 싶으신가요?

## B. 리소스 조사 요청 (IT담당자용)

### 문서/파일
- [ ] FAQ 문서 전체 (파일 형식, 저장 위치)
- [ ] 상담 매뉴얼/가이드 목록 (배송 절차, 클레임 처리, 요금 안내 등)
- [ ] 각 문서의 대략적 건수
- [ ] 샘플 문서 3~5건

### 시스템/DB
- [ ] 상담 이력 시스템 (CRM, 상담 로그 DB 등)
- [ ] 접근 방식 (DB 직접 / API / 파일 내보내기)
- [ ] 상담 이력 규모 (건수, 보관 기간)

### 인프라
- [ ] 현재 사용 중인 AWS 서비스 목록
- [ ] 상담 데이터가 어디에 저장되어 있는지 (S3, RDS 등)

### 보안/접근
- [ ] 상담 내용에 개인정보(이름, 전화번호, 주소) 포함 여부
- [ ] 고객 데이터 처리 관련 내부 규정 유무

## 1. 회사 개요
- 업종: 택배 물류 (B2C)
- 규모: 직원 200명 (CS팀 상담사 15명)
- 주력 사업: 택배 배송, 일 평균 3만건

## 2. AI/LLM 현황
- 기존 시도: 규칙 기반 챗봇 도입 후 3개월 만에 철거 (실패)
- 남은 자산: 없음

## 3. 인프라
- 환경: AWS (EC2, RDS, S3), Bedrock 가능
- DB: PostgreSQL(RDS), CRM REST API
- GPU: 불필요

- 개인정보처리방침: AI 조항 추가 정비 권고

## 5. 원하는 것
- 유즈케이스: CS 상담 보조 (내부 도구)
- 니즈: 반복 질문(60%) 응대 시간 단축
- AI 역할: FAQ/매뉴얼 검색 + 답변 초안 제시, 최종은 상담사
- 향후: 고객 챗봇, 물류센터, 셀러 횡전개

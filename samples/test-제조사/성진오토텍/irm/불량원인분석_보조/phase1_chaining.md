> 작성일: 2026-04-24

## 체이닝 패턴

| # | 질의 | 체이닝 순서 | GraphRAG | 신뢰도 |
|---|------|-----------|:--------:|:------:|
| 1 | "이 불량의 원인을 분석해줘" | 접근→접근→탐색→탐색→탐색→실행(유연) | Y | ⚠️ |
| 2 | "8D 보고서 초안 작성해줘" | (패턴1 결과)→탐색→실행(유연) | N | ⚠️ |
| 3 | "피로파괴와 크랙이 같은 건 맞아?" | 접근→탐색(GraphRAG) | Y | ✅ |

## 체이닝 플로우 상세

### 패턴 1: 불량 원인 분석 (핵심 체이닝 — 6단계)

가장 복잡한 체이닝. 이 유즈케이스의 메인 플로우.

```
[입력] 불량 건 ID 또는 불량 정보
    │
    ▼
① [접근] cli-mes: 불량 건 상세 조회
    │   → 불량코드, 라인, 제품, 일시, 불량 내용
    │
    ▼
② [접근] cli-defectcode: 불량코드 확장
    │   → "B-crack" → ["B-crack", "브레이크디스크_균열"] (타 공장 코드 포함)
    │
    ▼
③ [탐색] defect_history + GraphRAG 질의 확장
    │   → 확장된 코드로 유사 불량 이력 검색
    │   → GraphRAG: DefectType→CAUSED_BY→RootCause 관계 탐색
    │   → 결과: 유사 불량 N건 + 과거 원인 패턴
    │
    ▼
④ [탐색] quality_standards
    │   → 해당 부품/공정의 검사기준서 + 작업표준서 검색
    │   → 결과: 관련 기준/표준 문서
    │
    ▼
⑤ [탐색] investigation_reports
    │   → 유사 건의 과거 8D 보고서 검색
    │   → 결과: 과거 원인-대책 패턴
    │
    ▼
⑥ [실행(유연)] LLM 종합 분석
    │   → ①~⑤ 결과를 종합하여 원인 후보 제시
    │   → 추천 원인 + 근거 + 신뢰도
    │
    ▼
⑦ [Human-in-the-loop] 품질 엔지니어 확정
```

**의존성:**
- ②는 ①의 불량코드에 의존
- ③은 ②의 확장된 코드에 의존
- ④⑤는 ①의 부품/공정 정보에 의존 (③과 병렬 가능)
- ⑥은 ③④⑤ 모두 완료 후

**병렬화 가능 구간:**
```
① → ② → ③ ──┐
         ④ ──┤→ ⑥ → ⑦
         ⑤ ──┘
```
④⑤는 ③과 병렬 실행 가능 (①의 결과만 있으면 됨).

### 패턴 2: 8D 보고서 생성 (패턴 1 후속)

```
[입력] 패턴 1의 확정된 원인 + 관련 자료
    │
    ▼
① [탐색] investigation_reports
    │   → 유사 건의 과거 8D 보고서 (문체/구조 참조)
    │
    ▼
② [실행(유연)] LLM 8D 초안 생성
    │   → D1~D8 항목별 자동 생성
    │
    ▼
③ [Human-in-the-loop] 품질 엔지니어 검토/수정/승인
```

### 패턴 3: 용어 관계 확인 (단순 체이닝)

```
[입력] "피로파괴와 크랙이 같은 건 맞아?"
    │
    ▼
① [접근] cli-defectcode: 두 코드 매핑 확인
    │
    ▼
② [탐색] GraphRAG: DefectCode→MEANS→DefectType 관계 확인
    │   → 같은 DefectType을 가리키는지 확인
    │
    ▼
③ [응답] 관계 설명 + 근거 (어떤 공장/제품군에서 어떻게 불리는지)
```

## 전체 플로우 (End-to-End)

실제 사용 시나리오: 불량 발생 → 원인 분석 → 8D 보고서까지 한 번에.

```
[불량 발생]
    │
    ▼
패턴 1: 원인 분석 (6단계 체이닝)
    │   → 품질 엔지니어 원인 확정
    │
    ▼
패턴 2: 8D 보고서 생성 (패턴 1 결과 활용)
    │   → 품질 엔지니어 승인
    │
    ▼
[완료] 원인 분석 + 8D 보고서
```

**예상 소요 시간 (AI 보조 후):**
- 패턴 1: 10~15분 (현재 3~4시간 → 80% 단축)
- 패턴 2: 5~10분 (현재 30분~1시간 → 70% 단축)
- **합계: 15~25분 (현재 반나절 → 80% 단축 예상)**

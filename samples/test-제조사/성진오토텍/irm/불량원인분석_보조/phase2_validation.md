## 행위 유형별 검증

### 탐색 검증
| 항목 | 기준 | 측정 방법 |
|------|------|----------|
| 검색 정확도 (RAG) | 관련 문서 top-5 중 3건 이상 적중 (60%) | 테스트 질의 20개, 전문가 판정 |
| GraphRAG 질의 확장 | "크랙" 검색 시 "B-crack", "브레이크디스크_균열" 모두 포함 | 매핑 엑셀 60쌍 전수 테스트 |
| 응답 시간 | 검색 + 리랭킹 3초 이내 | 평균 응답 시간 측정 (20회) |
| 컬렉션 커버리지 | 모든 소스 인덱싱 완료 | 건수 대조 (원본 vs 인덱싱) |
| GraphRAG 인과 추적 | 2-hop 경로 정상 반환 | DefectType→RootCause 경로 10건 테스트 |

### 실행 검증
| 항목 | 기준 | 측정 방법 |
|------|------|----------|
| 원인 분석 추천 적중률 | 추천 top-3 중 1건 이상 엔지니어 수용 (70%) | 파일럿 20건 |
| 8D 보고서 초안 품질 | 엔지니어 수정률 30% 이하 | 파일럿 10건, 수정 분량 측정 |
| 체이닝 성공률 | 7단계 파이프라인 정상 완료 (95%) | 파일럿 기간 전수 |
| 엔드투엔드 소요 시간 | 30분 이내 (현재 반나절) | 파일럿 20건 평균 |

## 엔드투엔드 검증

### 체이닝 시나리오
| # | 시나리오 | 체이닝 | 기대 결과 | 합격 기준 |
|---|---------|-------|----------|----------|
| 1 | 화성공장 브레이크 디스크 크랙 불량 | 접근→접근→탐색x3→실행 | 유사 이력 + 기준서 + 원인 추천 | top-3 원인 중 1건 적중 |
| 2 | 아산공장 코드로 검색 ("브레이크디스크_균열") | 접근→GraphRAG확장→탐색 | 화성공장 "B-crack" 이력도 포함 | 교차 검색 성공 |
| 3 | 원인 분석 → 8D 보고서까지 | 패턴1→패턴2 | 원인 확정 + 8D 초안 | 30분 이내 완료 |
| 4 | "피로파괴와 크랙 같은 건지" 용어 확인 | 접근→GraphRAG | 동의어 관계 설명 | MEANS 관계 정상 반환 |
| 5 | severity=critical 불량 장기 이력 검색 | 탐색 | 3년 이상 이력 포함 | 기간 무관 전수 히트 |

## PoC 범위
- 대상: 화성공장 브레이크 디스크 라인
- 기간: 2주 운영 (파일럿 마지막 2주)
- 참여: 품질 엔지니어 3~5명
- 성공 기준:
  - 원인 분석 소요 시간 60% 이상 단축
  - 원인 추천 적중률 70% 이상
  - 8D 보고서 수정률 30% 이하
  - 사용자 만족도 4/5 이상
- 확대 조건: 성공 기준 충족 시 아산공장 + 서스펜션 라인 확대

## 분포 해석
- 주요 유형: **탐색(내부) 중심 (50%)** — 문서 검색 + 이력 검색이 핵심
- 접근(25%): MES 직접 조회 — CLI로 해결
- 실행(유연)(25%): 원인 분석 판단 + 8D 보고서 생성 — LLM + Human-in-the-loop
- 실행(확정): 없음 — 모든 판단이 상황 의존적
- 아키텍처 방향 시사: **RAG 투자 중심 + CLI 보조 + LLM 판단**
- **분포 신뢰도**: ✅확정 67% / ⚠️추정 33% — 8D 보고서 위치 확인 후 추정 비율 감소 예상

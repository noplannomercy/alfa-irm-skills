# 불량원인분석_보조 — Phase 1 종합 요약

> 작성일: 2026-04-24
> 고객: 성진오토텍

## 아키텍처 개요

### 행위 유형별 배치

| 행위 유형 | 비율 | 배치 방식 | 핵심 구성요소 | 신뢰도 |
|----------|------|----------|-------------|:------:|
| 탐색 | 54% | RAG + GraphRAG | 3 컬렉션, pgvector, BGE-m3-ko, Apache AGE | ✅ |
| 접근 | 31% | CLI + REST | cli-mes, cli-defectcode, FastAPI | ✅ |
| 실행(유연) | 15% | LLM + Human-in-the-loop | 원인 분석 보조, 8D 보고서 생성 | ⚠️ |

**설계 신뢰도:** ✅ 67% / ⚠️ 33% — 8D 보고서 위치/건수 미확인이 주요 원인

### GraphRAG 판단
- 결정: **승인 (3/4 충족)**
- 핵심: 불량코드 30% 불일치(매핑 엑셀 시드) + 인과관계 질의 23% + 동일 엔티티 다중 맥락

### 인프라 요약
- LLM: 로컬 (Ollama) — RTX 4090 x2, 모델 미정 (온프레미스 필수)
- 임베딩: BGE-m3-ko (로컬)
- 리랭커: BGE-reranker-v2-m3 (로컬)
- 벡터DB: pgvector (PostgreSQL 16, HNSW)
- 그래프DB: Apache AGE (같은 PostgreSQL 16)
- REST: FastAPI (내부 전용)
- CLI: 2개 (cli-mes, cli-defectcode)

### 기존 인프라 재사용
- 신규 구축: 전부 (기존 AI 인프라 없음)
- 다른 유즈케이스 공유 가능:
  - pgvector + Apache AGE → 향후 "협력사 품질 평가" 유즈케이스에서 재사용
  - quality_standards 컬렉션 → 다른 품질 관련 유즈케이스에서 공유 가능
  - cli-mes → MES 조회가 필요한 모든 유즈케이스에서 재사용

## 예상 일정

| 단계 | 기간 | 내용 | 리스크 가산 |
|------|------|------|-----------|
| 인프라 | 1주 | GPU 수령 + Ollama + PostgreSQL(pgvector+AGE) 세팅 | — |
| 탐색(RAG) — 기준서 | 1.5주 | quality_standards 컬렉션: PDF 파싱 + HWP 파싱 테스트 + 임베딩 + 인덱싱 (500건) | HWP 파싱 +0.5주 |
| 탐색(RAG) — 불량이력 | 1주 | defect_history 컬렉션: MES 데이터 전처리 + 임베딩 (15만건 이하) | — |
| 탐색(RAG) — 8D보고서 | 0.5주 | investigation_reports 컬렉션: 위치 확인 후 인덱싱 | 위치 미확인 +0.5주 |
| GraphRAG | 2주 | Apache AGE 스키마 + 매핑 엑셀 시드(60쌍) + 8D보고서 트리플 추출 | 복잡도 높음 +1주 |
| 접근(CLI) | 1주 | cli-mes + cli-defectcode 구현 + 단위 테스트 | — |
| 실행(유연) | 1.5주 | LLM 원인 분석 프롬프트 설계 + 8D 보고서 생성 + Human-in-the-loop UI | 판단 복잡도 +0.5주 |
| 통합 테스트 | 1.5주 | 체이닝 3패턴 엔드투엔드 + 감사로그 + 보안검토서 제출 | 체이닝 3개 +0.5주 |
| **합계** | **12주** | | 리스크 가산 포함 |

### 일정 타임라인 (8월 말 마감 대비)

```
5월 초   ← GPU 입고 + 인프라 세팅
5월 중   ← RAG 컬렉션 3개 구축 (3주)
6월 초   ← GraphRAG 구축 (3주)
6월 말   ← CLI 구현 + 실행(유연) 설계 (2.5주)
7월 중   ← 통합 테스트 + 보안검토 (2주)
8월 초   ← 파일럿 운영 (2주)
8월 말   ← 결과 보고
```

**보안검토 2주는 통합 테스트와 병렬 처리 가능** (보안검토서 제출 → 승인 대기 중 테스트 진행)

### 크리티컬 패스
1. GPU 입고 (5월 초) — 모든 작업의 시작
2. HWP 파싱 테스트 (5월 중) — 품질 불량 시 대안 필요
3. 8D 보고서 위치 확인 — investigation_reports 컬렉션 + GraphRAG 트리플 추출에 영향

### 예산 적합성
- 파일럿 예산: 3천만원
- 주요 비용: GPU(이미 승인, 회사 자산), 외부 인력 12주, 소프트웨어(전부 오픈소스)
- 판단: **적합** — 외부 인력 비용이 주요 항목, 3천만원 내 가능

# 성진오토텍 ALFA 입력 프로필 (확정)

> 확정일: 2026-04-24
> 버전: v1.0
> 근거 미팅: 1차(4/10), 2차(4/17), 3차(4/24)

## 1. 회사 개요
- 업종: 자동차 부품 제조
- 규모: 직원 800명 (품질관리 인력 약 60명)
- 주력 사업: 브레이크 디스크, 서스펜션 부품
- 납품처: 현대기아 1차 벤더
- 사업장: 본사 + 화성공장 + 아산공장

## 2. AI/LLM 현황
- 기존 시도: 없음 (LLM 경험 전무)
- 남은 자산: 비전 검사(카메라 기반) 화성공장 운영 — LLM과 무관

## 3. 인프라
- 환경: 온프레미스 only (사내 서버)
- DB: MES(미라콤) ODBC 직접 조회, REST API 없음. ERP(SAP).
- GPU: RTX 4090 x2 — 구매 승인, 2주 내 입고
- 파일서버: \\filesvr\품질관리\
- 네트워크: 외부 클라우드 전송 금지 (OEM 보안 가이드라인)

## 4. 규제 환경
- 감독기관: 없음 — OEM(현대기아) 요구사항이 사실상 규제
- 보안: 현대기아 협력사 정보보안 가이드라인, 품질 데이터 "대외비"
- 컴플라이언스: 경영지원팀 최유진 겸임 (별도 팀 없음)
- 시스템 도입: 보안검토서 → 경영지원팀 승인 (2주)
- AI 요건: 학습 데이터 사용 금지 (로컬 LLM 충족), 감사로그 필수
- IATF 16949: AI 보조도구 별도 인증 불필요, 품질매뉴얼에 사용 기준 추가 권고

## 5. 원하는 것
- 유즈케이스: 불량 원인 분석 보조 (화성공장 브레이크 디스크 라인)
- 현업 니즈: 불량 1건 원인 분석에 반나절 소요 → 시간 단축
- 프로세스: 불량접수 → 이력검색(병목) → 기준서확인(병목) → 원인분석 → 8D보고서
- AI 역할: 8D 보고서 초안까지 보조, 최종 판단은 사람
- 보류: 협력사 품질 평가 (파일럿 성공 후 2차)

## 6. 제약사항
- 예산: 파일럿 3천만원, 효과 시 내년 1억
- 일정: 8월 말까지 파일럿 결과 보고 (약 4개월)
- 인력: IT팀 이정수 과장 1명 파트타임
- 대면: 내부 직원만 (납품처 노출 없음)
- 보안검토: 도입 전 2주 소요

| 역할 | 담당 |

## 데이터 분류 및 처리
- 분류 체계: 품질 데이터 = "대외비" 등급
- 개인정보: 포함 안 됨
- OEM 영업비밀: 가능성 있음 — 대외비로 관리
- LLM 투입 가능 범위: **사내 서버에서만 처리** (외부 클라우드 전송 금지)
- 학습 데이터 사용: **금지** → 로컬 LLM으로 충족

## 접근 제어
- 기존 체계: MES/SAP 기존 권한 체계
- AI 추가: **감사로그 필수** — 누가 어떤 데이터를 조회했는지 기록
- 시스템 도입: 보안검토서(IT팀 작성) → 경영지원팀 승인 (2주 소요)

## 기술 신호
- 거버넌스 무게: **Medium** — 근거: 제조업, 내부 직원 한정, 규제기관 없음. OEM 보안요구로 데이터 반출 제한.
- GraphRAG 필요: **예 (강한 신호)** — 신호: "같은 불량인데 화성공장이랑 아산공장이 다르게 기록해서 분석이 안 된다" → 공장별 용어/코드 불일치 확인
- 인프라 경로: **온프레미스 로컬 LLM** — 현대기아 보안 요구, 클라우드 미사용, GPU 없음 → GPU 세팅 필요
- PoC 이력: 없음 (LLM 경험 전무, 비전 검사만 별도 운영)

## 감지된 리스크
- **GPU 부재**: 온프레미스 확정인데 GPU가 없음. 조달 1~2개월 소요 가능 → 일정 리스크
- **문서 형식 혼재**: PDF + 한글(HWP) → 파싱 난이도 상승, 한글 파서 필요
- **예산 미확정**: "올해 효과 보여주면 내년 예산" → 파일럿 비용을 누가 부담하는지 불명확
- **공장 간 데이터 불일치**: 화성/아산 불량 코드 다름 → 데이터 정제 or GraphRAG로 대응

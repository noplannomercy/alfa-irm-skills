# 성진오토텍 2차 미팅 서머리

> 일시: 2026-04-17
> 참석: 김태호 CTO, 박성민 품질관리팀장, 이정수 IT팀 과장, 당사 2명

## 논의 내용

1. **대상 업무 선정**: 후보 3개(불량 원인 분석, 검사기준서 검색, 협력사 품질 평가) 중 **"불량 원인 분석 보조"**를 파일럿 대상으로 확정. 가장 시간 소요가 크고 효과가 가시적인 업무.
2. **현재 프로세스 확인**: 불량 접수 → 유사 이력 검색 → 기준서 확인 → 원인 분석 → 8D 보고서 작성의 5단계. 특히 2단계(이력 검색)와 3단계(기준서 탐색)에서 병목 발생.
3. **데이터 접근 확인**: MES는 ODBC로 직접 조회 가능. 품질 문서는 사내 파일서버에 정리됨.
4. **인프라 업데이트**: 딥러닝용 GPU(RTX 4090 2장) 구매 품의 진행 중.
5. **파일럿 범위**: 화성공장 브레이크 디스크 라인으로 한정.
6. **AI 역할 합의**: 8D 보고서 초안까지 AI가 보조하되, 최종 판단과 승인은 품질 엔지니어가 수행.

## 합의 사항
- 파일럿 대상: **불량 원인 분석 보조** (화성공장 브레이크 디스크 라인)
- 예산: 파일럿 3천만원 이내
- 일정: 8월 말까지 파일럿 결과 보고
- IT팀 1명(이정수 과장) 파트타임 투입
- 검사기준서 검색은 불량 분석 과정에 자연스럽게 포함

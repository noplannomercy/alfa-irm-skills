# 성진오토텍 3차 미팅 메모

> 일시: 2026-04-24
> 참석: 김태호 CTO, 박성민 품질관리팀장, 이정수 IT팀 과장, 최유진 경영지원팀 (보안담당), 우리 측 2명

## 미팅 내용 (원문)

- 이번 미팅은 OEM 보안 요구사항 확인 + 유즈케이스 최종 합의가 목적

- 보안담당 최유진:
  - 현대기아 협력사 정보보안 가이드라인 적용 중
  - 품질 데이터는 "대외비" 등급. 개인정보는 포함 안 됨
  - 외부 클라우드 전송 금지 (사내 서버에서만 처리 가능)
  - 새로운 시스템 도입 시 보안검토 필요 — IT팀이 보안검토서 작성하면 경영지원팀 승인 (2주 소요)
  - AI 모델이 학습에 데이터를 사용하지 않는다는 보장 필요 → 로컬 LLM이면 문제없음
  - 감사로그 필요: 누가 어떤 데이터를 조회했는지 기록
  - IATF 16949 관련: AI 도입이 기존 품질경영시스템에 영향 없으면 별도 인증 불필요. 다만 내부 품질매뉴얼에 AI 보조도구 사용 기준 추가 권고

- 유즈케이스 재확인:
  - 불량 원인 분석 보조 → 그대로 진행 합의
  - CTO가 "협력사 품질 평가도 같이 하면 안 되나?" 요청
  - 우리 측: "파일럿은 하나에 집중해야 효과 측정이 명확합니다. 불량 분석으로 인프라 깔아놓으면 협력사 평가는 2차로 빠르게 확장 가능합니다" 설명
  - CTO 수긍: "그래, 우선 하나 제대로 하자"

- 품질 문서 샘플 수령 확인:
  - 검사기준서 PDF 5건, 작업표준서 HWP 3건 수령 완료
  - MES 불량 이력 샘플: 화성공장 최근 6개월 약 8,000건 CSV로 수령

- 불량코드 관련:
  - IT팀 이정수 과장이 화성/아산 불량코드 매핑 엑셀을 정리해옴
  - 약 200개 불량코드 중 30% 정도가 공장 간 불일치 확인됨
  - "예전에 통일하려고 했는데 현장에서 바꾸기 싫어해서 포기했다"고 함

- GPU 현황: 품의 승인됨, 2주 내 입고 예정

- 컴플라이언스팀: 별도 팀은 없음. 경영지원팀 최유진이 보안/컴플라이언스 겸임.

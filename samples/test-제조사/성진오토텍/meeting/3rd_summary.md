# 성진오토텍 3차 미팅 서머리

> 일시: 2026-04-24
> 참석: 김태호 CTO, 박성민 품질관리팀장, 이정수 IT팀 과장, 최유진 경영지원팀(보안담당), 당사 2명

## 논의 내용

1. **보안 요구사항 확인**: 경영지원팀 보안담당을 통해 데이터 관리 기준을 확인. 품질 데이터는 사내 서버에서만 처리하며, 시스템 도입 시 보안검토(약 2주)를 거쳐 승인.
2. **파일럿 대상 최종 확인**: "불량 원인 분석 보조"를 파일럿 대상으로 최종 합의. 협력사 품질 평가는 파일럿 성공 후 2차 확장으로 진행하기로 함.
3. **샘플 데이터 수령**: 검사기준서 5건, 작업표준서 3건, MES 불량 이력 화성공장 6개월분(약 8,000건) 수령 완료.
4. **불량코드 현황**: 화성/아산 공장 간 불량코드 매핑 자료 수령. 약 200개 코드 중 30% 정도 표현 차이가 있어 이를 고려한 설계 필요.
5. **GPU 확보**: 구매 승인 완료, 2주 내 입고 예정.
6. **품질 인증**: AI 보조도구 도입이 기존 IATF 16949 인증에 영향 없음 확인. 내부 품질매뉴얼에 사용 기준 추가 권고.

## 합의 사항
- 파일럿 대상: 불량 원인 분석 보조 (최종 확정)
- 보안검토서 제출 후 경영지원팀 승인 필요 (도입 전)
- 데이터 조회 이력 기록(감사로그) 포함 설계
- AI는 보조 역할, 최종 판단은 품질 엔지니어가 수행 (기존 합의 유지)

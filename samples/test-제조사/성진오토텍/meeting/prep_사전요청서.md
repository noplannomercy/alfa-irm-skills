## A. 사전 질의 (담당자용)

아래 항목 중 현재 파악 가능한 내용을 정리해주세요.

### 현황 관련
- 주력 제품군과 주요 납품처가 어떻게 되시나요?
- 현재 품질관리 업무에서 가장 시간이 많이 걸리거나 어려운 부분이 무엇인가요?
- 이전에 AI 또는 데이터 분석 관련 프로젝트를 진행하신 적이 있으신가요? (결과 포함)
- 품질검사 데이터(검사 결과, 불량 이력 등)는 어떤 시스템에서 관리하고 계신가요?

### 목표/기대 관련
- AI 도입으로 기대하시는 구체적인 효과가 있으시면 알려주세요
- 예산/일정 계획이 잡혀있으시면 공유 부탁드립니다
- 이 프로젝트의 내부 의사결정 담당자(스폰서)가 누구인지 알려주세요

## B. 리소스 조사 요청 (IT팀용)

AI 도입 범위를 구체화하기 위해 아래 정보가 필요합니다.
미팅 전 또는 미팅 후 1주 내 회신 부탁드립니다.

### 문서/파일
- [ ] 품질 매뉴얼, 검사 기준서, 작업 표준서 등 관련 문서 목록
- [ ] 각 문서의 저장 위치, 파일 형식, 대략적 건수
- [ ] 샘플 문서 3~5건 (내용 확인용)

### 시스템/DB
- [ ] 품질 관련 시스템 목록 (MES, ERP, SPC, 검사장비 등)
- [ ] 각 시스템의 접근 방식 (DB 직접 / REST API / 파일 내보내기 등)
- [ ] DB 테이블 구조 또는 ERD (해당 시)
- [ ] 품질 데이터 규모 (검사 건수, 불량 이력 건수, 보관 기간)

### 절차/매뉴얼
- [ ] 품질검사 절차서 목록 (인수검사, 공정검사, 출하검사 등)
- [ ] 불량 발생 시 처리 프로세스 (원인 분석 → 대책 수립 단계)
- [ ] 각 절차의 단계 수, 판단 기준 유무

### 보안/접근
- [ ] 품질 데이터에 대한 보안 등급이나 접근 제한 사항
- [ ] 데이터 외부 반출 가능 여부 (클라우드 전송 가능 여부)
- [ ] 납품처(OEM)의 데이터 관리 요구사항 유무

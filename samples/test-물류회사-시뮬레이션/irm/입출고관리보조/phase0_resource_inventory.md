## 리소스 목록

| # | 리소스명 | 유형 | 위치/접근 방법 | 비고 |
|---|---------|------|--------------|------|
| 1 | 입출고 매뉴얼 (센터별) | 문서/파일 | 사내 문서 서버 | 5개 센터별 약간 다름 |
| 2 | 이상 처리 절차서 | 절차/매뉴얼 | 사내 문서 서버 | 파손/수량불일치/오배송/유통기한/라벨오류 |
| 3 | WMS 입출고 이력 | 시스템 | MySQL (RDS) | 바코드 스캔 기반, 이상 건 포함 |

| 5 | 이상 건 처리 이력 | 시스템 | MySQL (RDS) | 과거 이상 건 처리 결과 (자유 텍스트 기록) |
| 6 | 검수 기준표 | 문서/파일 | 사내 문서 서버 | 품목별 검수 항목/기준 |

## 리소스 유형별 요약
- 문서/파일: 2건 (매뉴얼, 검수 기준표)
- 시스템: 3건 (WMS 입출고/재고/이상이력)
- 절차/매뉴얼: 1건 (이상 처리 절차서)
- **합계: 6건**

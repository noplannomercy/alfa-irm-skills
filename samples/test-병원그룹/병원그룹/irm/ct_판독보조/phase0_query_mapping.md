## 체이닝 패턴 (복합 질의)

복수 행위 유형을 연결하는 질의:

- **"이 소견과 비슷한 판독문 찾아서 초안 만들어줘"**
  = 탐색(내부, 유사 판독문 검색) → 실행(유연, 초안 생성)
  → 관련 리소스: #1, #4, #5

- **"이 환자의 이전 판독 이력 보고, 이번 소견과 비교해줘"**
  = 접근(EMR/PACS 이력 조회) → 탐색(내부, 유사도 비교) → 실행(유연, 비교 요약)
  → 관련 리소스: #1, #2, #3

- **"새 판독문 작성하고 가명처리까지 해줘"**
  = 실행(유연, 판독문 초안) → 실행(확정, 가명처리) → 실행(확정, 감사로그 기록)
  → 관련 리소스: #1, #5, #6, #7

## Phase 0 결론

### 행위 유형 분포 요약
- **주요 유형**: 접근(29%) + 탐색(35%) = 64% — 정보 검색·조회 중심
- **아키텍처 시사**: 탐색 품질이 유즈케이스 성패 결정. 특히 의미 기반 검색(동의어/유사 표현 매핑)이 핵심. 실행(유연)은 탐색 결과를 기반으로 동작하므로 탐색 정확도가 전체 파이프라인 품질을 좌우

### GraphRAG 사전 신호 (Phase 1에서 정식 판단)
- 동음이의어/용어 불일치: **있음** — "간 종괴" = "hepatic mass" = "liver mass" = "간 SOL", 의사마다 같은 소견을 다르게 기술
- 인과관계 질의 비율: 2/14 (14%) — "비슷한 소견의 다른 의사 판독 스타일 비교", 소견-진단 관계 추적
- 동일 엔티티 다중 맥락: **있음** — 같은 병변이 시간축(이전 판독 vs 현재), 의사축(의사별 표현), 용어축(한글/영문/약어)으로 다중 표현
- → 사전 판단: **후보** (강함) — 3개 신호 모두 해당, 이전 PoC 실패 원인과도 직결

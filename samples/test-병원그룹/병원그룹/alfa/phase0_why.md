# 병원그룹 — Phase 0: WHY

> 작성일: 2026-03-25
> 상태: 확정

## 왜 하는가?
- 트리거: CTO 지시 — "AI 도입 검토하라"
- 배경: 영상의학과 판독 업무량 증가 + 의사마다 판독 표현이 달라 과거 판독문 검색이 비효율적. 작년 외부 업체 챗봇 PoC가 의학용어 이해 부족으로 실패 → 도메인 특화 접근 필요성 공감대 형성

## 스폰서
- CTO
- 의사결정 권한: 예산 승인 (PoC 5천만원), 인력 배정 (의료정보팀 3명), 일정 확정

## 기대 효과 (KPI)
- CT 판독문 작성 시간 단축 (기존 대비 측정 예정)
- 과거 유사 판독 사례 검색 정확도 향상 (동의어/유사 표현 포함 검색)
- 판독문 초안 품질 — 의사 최소 수정으로 사용 가능 수준

## 범위
- 1차 (PoC): 본원 CT 판독문 보조 — 유사 판독 검색 + 초안 생성 + 용어 표준화
- 향후: MRI 확장 → 간호기록 요약 → (장기) 환자 설명용 활용

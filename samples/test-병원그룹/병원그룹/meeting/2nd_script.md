# 병원그룹 2차 미팅 메모

> 일시: 2026-03-25
> 참석자: 의료정보보안팀장, IRB 담당자, 영상의학과 과장

## 미팅 내용 (원문)

[1차 미확인 → 이번에 확인]
- 병원 전체 규모: 본원 500병상, 분원 각 150~200병상, 연간 영상 촬영 약 12만건 (본원 기준)
- 이전 AI 시도: 작년에 외부 업체가 챗봇 PoC 했는데 정확도 낮아서 중단. "의학용어를 못 알아들어서 엉뚱한 답변이 나왔어요"
- PACS는 Infinitt 제품, 본원/분원 동일. HL7 FHIR 연동 가능 확인됨
- 정보보안 담당: 의료정보보안팀(3명) 별도 운영 중, 개인정보보호책임자는 행정부원장
- IT팀 투입: 의료정보팀 3명 투입 가능, 인프라 담당 1명 + 개발 2명

[규제/거버넌스 확정]
- 개인건강정보는 가명처리 후 LLM 투입 가능하다는 내부 검토 결과 나옴
- 가명처리 기준은 보건복지부 가이드라인, 이름/생년월일/주민번호 마스킹
- IRB 간편심의로 진행 가능 확인됨, 후향적 데이터라 2주면 통과 예상
- "판독문은 의료정보니까 외부 API 절대 안 됨, 로컬에서만 돌려야 해요"
- 접근권한은 기존 EMR 권한체계 따르기로, PACS 연동은 읽기전용만
- 의료정보보안팀장: "AI 결과를 의사가 최종 확인하는 구조면 보조수단으로 볼 수 있다"
- 감사로그는 기존 EMR 감사로그에 AI 조회 이력 추가하는 방식
- 데이터 보관은 원내 서버, 학습용 데이터셋은 별도 격리 스토리지

[스코핑/합의]
- "분원 데이터는 1차에선 빼고, 본원 데이터만으로 PoC 하자"
- 의사결정 기구는 기존 의료정보위원회에서 AI 관련 안건 심의하기로
- RACI: 실행=의료정보팀, 검토=영상의학과, 승인=의료정보위원회, 보고=CTO
- A100 연구팀 공유 건: 주중 업무시간 우선 사용 합의 완료
- 예산 확정: PoC 기준 5천만원 이내
- 영상의학과 과장: "CT 판독문 먼저 해보고, 잘 되면 MRI도 넣자"
- "나중에 환자 설명용으로도 쓸 수 있으면 좋겠는데 그건 먼 얘기고"
- PoC 시작은 4월 중순 목표, IRB 2주 + 데이터 준비 2주 감안

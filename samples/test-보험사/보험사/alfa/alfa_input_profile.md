## 1. 회사 개요
- 회사명: 미래에셋손해보험
- 업종: 손해보험 (금융)
- 규모: 중견 (직원 1,200명, 보험료 수입 연 2조)
- 주요 상품: 자동차/화재/배상책임/상해보험
- 본사 서울, 지점 12곳

## 2. AI/LLM 현황
- 기존 시도: OCR+RPA (2년 전) → 정확도 낮아서 중단
- 남은 자산: OCR 데이터 16만건 (S3, 품질 들쭉날쭉)

## 3. 인프라
- AWS (EC2, RDS, S3)
- DB: Oracle (코어) + PostgreSQL (심사, 5년 60만건) + MongoDB (문서)
- 시스템: 코어 + 심사 + 손해사정 (3개 별도, 각각 API/ERD 확인)

## 4. 규제 환경
- 금감원 대상, 7대 원칙 적용
- 보조수단성 충족 (AI 보조, 심사역 판단)
- 개인정보 가명처리 + 신용정보 LLM 투입 금지
- 컴플라이언스팀 3명, 준법감시인 = 개인정보보호책임자
- 판단 근거 로깅 필수

## 5. 원하는 것
- 심사 보조 (유사 사례 검색 + 약관 적용 안내) — 1순위
- 손해사정 서류검토 보조 — 2순위
- PoC: 자동차보험 심사 보조 (전체 심사의 45%)

## 질의별 행위 유형 분포

## 체이닝 패턴 (복합 질의)

- **"이 건 심사해줘" (전체 보조)**
  = 접근(계약 조회) → 탐색(유사 사례 + 약관 + 가이드라인) → 실행(유연, 의견 초안) → 실행(확정, 감사로그)

- **"이 특약 적용 가능한지 확인해줘"**
  = 탐색(약관 조항 검색, 특약 간 관계 추적) → 접근(계약 정보 확인) → 실행(유연, 적용 판단)

- **"거절된 건 재심사 해줘"**
  = 접근(원심사 이력) → 탐색(유사 재심사 사례) → 실행(확정, 재심사 절차)

## Phase 0 결론

### 행위 유형 분포 요약
- **주요 유형**: 탐색 57% — 압도적 탐색 중심
- **아키텍처 시사**: RAG 품질이 유즈케이스 성패 결정. 약관 관계 추적이 핵심.

### GraphRAG 사전 신호
- 동음이의어/용어 불일치: **있음** — "면책"이 일반면책/특별면책/상법면책으로 맥락 따라 다름, 심사역마다 기준 해석 차이
- 인과관계 질의 비율: 4/14 (29%) — "특약 A→B 관계", "할증 조건→보험료", "면책→특약 관계"
- 동일 엔티티 다중 맥락: **있음** — 같은 사고 유형이 약관/가이드라인/과거 심사에서 다르게 기술
- → 사전 판단: **후보 (강)** — 3/4 해당 (인과관계 29% > 20%)

# 러닝브릿지 1차 미팅 서머리

> 일시: 2026-03-27
> 참석: IT팀장, 학습운영팀장

## 논의 내용

1. **현재 상담 운영 현황 공유**
   - 학습 상담사 10명이 하루 약 500건의 수강생 문의를 수동 처리 중
   - 문의 유형: 수강 방법(30%), 강의 내용 질문(35%), 환불/변경(20%), 기술 문제(15%)
   - 상담사마다 답변이 달라지는 일관성 문제 확인

2. **AI 도입 방향 논의**
   - 1단계: 내부 상담사를 위한 보조 도구로 시작
   - 2단계: 추후 수강생이 직접 사용하는 서비스로 확장 검토
   - 기존 FAQ 150건(구글독스)을 기반으로 활용

3. **기술 환경 확인**
   - AWS 기반 플랫폼 운영 중 (EC2, RDS, S3)
   - 강의 콘텐츠 3,000개(영상+PDF 교안) 보유
   - 수강생 수강이력/진도/성적 데이터 3년치 축적

4. **투입 자원 논의**
   - 예산: 2,000만원
   - 일정: 2026년 내 파일럿 프로젝트 완료
   - 인력: IT팀 2명 투입 가능 (백엔드 1명, 인프라 1명)

## 합의 사항
- 1단계는 내부 상담사 보조 도구로 범위를 한정
- 기존 FAQ를 활용한 파일럿 프로젝트로 시작
- 수강생 개인정보 처리 범위는 추가 검토 필요

## 다음 스텝
| 항목 | 담당 | 기한 |
|------|------|------|
| FAQ 150건 데이터 정리 및 공유 | 학습운영팀 | 미팅 후 1주 |
| 파일럿 대상 강의 범위 선정 | 학습운영팀 | 미팅 후 1주 |
| 인프라 환경 상세 조사 (DB 스키마, API 현황) | IT팀 | 미팅 후 2주 |
| 개인정보 처리 범위 내부 검토 | 담당자 확인 필요 | 미팅 후 2주 |
| 파일럿 프로젝트 제안서 전달 | 당사 | 위 자료 수신 후 |

## 참고
- 강의 내용에 대한 질문(35%)이 가장 난이도가 높아, 파일럿에서 어느 범위까지 다룰지 별도 협의 필요
- 수강생 중 미성년자 포함 여부 확인 시 개인정보 처리 기준이 달라질 수 있음

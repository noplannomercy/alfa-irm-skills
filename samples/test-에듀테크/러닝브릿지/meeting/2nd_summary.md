# 러닝브릿지 2차 확인 서머리

> 일시: 2026-03-27
> 참석: IT팀 (이메일 회신)

## 확인 내용

1. **강의 자막 현황**
   - 강의 영상 자막(SRT 파일)이 3,000개 중 2,800개 보유
   - 자막 미보유 강의 200개는 별도 확인 필요

2. **기존 상담 이력**
   - Zendesk에 2년치 약 36만건의 상담 기록 보유
   - API를 통한 데이터 추출 가능

3. **환불/변경 규정**
   - PDF 문서(10페이지) 공유 완료

4. **개인정보 및 보안 관련**
   - 별도 컴플라이언스팀 없음 (필요 시 법무 외주)
   - 미성년자 수강생 약 15%, 법정대리인 동의 수집 중
   - 개인정보 처리방침 홈페이지 게시 중, 내부 정책 문서 있음
   - 비식별화 방법: 이름 마스킹, 이메일 도메인만 노출 가능

## 합의 사항
- 파일럿 프로젝트에 필요한 주요 데이터 현황이 확인됨
- 개인정보 비식별화 방법이 확인되어 데이터 활용 범위 설정 가능

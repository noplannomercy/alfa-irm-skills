## 데이터 분류 및 처리
- 분류 체계: 수강생 개인정보 (이름/이메일/수강이력) → 개인정보보호법 대상
- 미성년자: 전체 수강생의 약 15%, 법정대리인 동의 수집 중 [2차 확인]
- 비식별화/가명처리: 이름 마스킹 + 이메일 도메인만 노출 가능 [2차 확인]
- 개인정보 처리방침: 홈페이지 게시 중, 내부 정책 문서 보유 [2차 확인]
- LLM 투입 가능 범위:
  - FAQ 150건: 개인정보 미포함 → 투입 가능
  - 강의 교안(PDF): 개인정보 미포함 → 투입 가능
  - 강의 자막(SRT): 개인정보 미포함 → 투입 가능 [2차 확인]
  - 환불/변경 규정(PDF): 개인정보 미포함 → 투입 가능 [2차 확인]
  - 수강생 데이터(이력/진도): 개인정보 포함 → 비식별화(마스킹) 후 투입
  - 상담 이력(Zendesk 36만건): 수강생 식별 정보 포함 가능 → 비식별화 후 투입 [2차 확인]

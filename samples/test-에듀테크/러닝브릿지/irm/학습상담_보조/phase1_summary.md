## 인프라 요약

| 구성요소 | 선택 | PoC 비용 (월) | 비고 |
|---------|------|:----------:|------|
| LLM | AWS Bedrock (Claude) | 50~100만 | 답변 생성 + 패턴 분석 |
| 임베딩 | OpenAI text-embedding-3-small | 5~10만 | PoC → BGE-m3-ko(로컬) 전환 |
| 벡터DB | pgvector (기존 RDS 확장) | 추가 비용 최소 | 기존 PostgreSQL 활용 |
| 리랭킹 | FlashRank (CPU) | 무료 | PoC → BGE-reranker-v2-m3 전환 |
| REST Gateway | FastAPI | — | EC2 내 배포 |
| CLI | 2개 (student, zendesk) | — | Python, 로컬 실행 |
| **월 인프라 합계** | | **~80~120만** | |

### 기존 인프라 재사용
- **RDS PostgreSQL**: 기존 수강생 DB + pgvector 확장 → 신규 DB 불필요
- **EC2**: 기존 서버에 FastAPI + CLI 배포 가능
- **S3**: 기존 강의 콘텐츠 저장소 그대로 활용
- **Zendesk**: 기존 CS 시스템 API 연동

### 신규 구축 필요
- pgvector extension 설치 (RDS에서 활성화)
- 3개 컬렉션 테이블 생성 + 인덱싱
- FastAPI Gateway 서비스 배포
- CLI 2개 패키지 설치
- 비식별화 파이프라인 (이름 마스킹, 이메일 도메인)

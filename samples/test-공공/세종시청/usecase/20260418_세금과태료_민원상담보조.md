# 유즈케이스 정의서: 세금/과태료 민원 상담 보조

> 고객: 세종시청
> 작성일: 2026-04-18
> 상태: 확정

## 개요
- 목적: 세금/과태료 관련 반복 민원에 대해 상담사가 빠르게 관련 정보를 찾고 답변할 수 있도록 보조
- 사용자: 민원상담팀 상담사 12명 (파일럿 5명)
- 입력: 시민 민원 내용 (전화 상담 중 상담사가 입력)
- 출력: 관련 FAQ + 매뉴얼 + 법령 검색 결과 + 답변 초안

## 행위 유형 분포 (예상)

| 행위 유형 | 비중 | 설명 |
|----------|------|------|
| 탐색(내부) | 50% | FAQ 검색, 매뉴얼 검색, 민원 이력 검색 |
| 탐색(외부) | 10% | 법제처 API — 지방세법 조항 조회 |
| 접근 | 20% | 새올 민원 조회, 세외수입 납부 현황 조회 |
| 실행(유연) | 20% | 답변 초안 생성 (LLM 판단) |
| 실행(확정) | 0% | 해당 없음 |

## 리소스 현황

| # | 리소스명 | 유형 | 근거 | 위치/접근 방법 | 규모 |
|---|---------|------|:----:|-------------|------|
| 1 | 세무과 FAQ | 문서/파일 | ✅ | 시청 홈페이지 DB | 약 150건 |
| 2 | 세무 매뉴얼 | 문서/파일 | ✅ | 공유 드라이브, HWP | 약 30건 |
| 3 | 지방세법 법령 | 외부소스 | ✅ | 법제처 API (공개) | 관련 법률/시행령 |
| 4 | 세금 민원 이력 | 시스템(새올) | ✅ | DB 직접 조회 | 약 3만건/년 |
| 5 | 부과/납부 데이터 | 시스템(세외수입) | ✅ | DB 직접 조회 | 개인정보 포함 — 비식별화 필수 |

**리소스 신뢰도: ✅5건 (100%)**

  - 감면 유형: "세금 깎아달라" → 장애인/다자녀/국가유공자 감면 (조건 다름)

## 체이닝 플로우 상세

### 패턴 1: 민원 답변 초안 (메인 플로우 — 7단계)

```
[입력] 시민 문의 내용
    │
    ▼
① [접근] cli-saeol: 민원 상세 (비식별화)
    │
    ▼ (선택적)
② [접근] cli-tax: 부과/납부 현황 (비식별화)
    │
    ▼ (병렬)
③ [탐색] tax_knowledge + GraphRAG 용어 확장
④ [탐색] tax_consultation: 유사 과거 응대
⑤ [탐색] cli-law: 관련 법령 (해당 시)
    │
    ▼
⑥ [실행(유연)] LLM 답변 초안 + 근거 표시
    │
    ▼
⑦ [Human-in-the-loop] 상담사 검토/수정/답변
```

병렬화: ③④⑤ 병렬 가능

### 패턴 2: 감면 유형 탐색 (GraphRAG 활용)

```
[입력] "세금 깎아달라"
    │
    ▼
① [GraphRAG] "감면" → 장애인/다자녀/국가유공자 등 하위 유형 확장
    │
    ▼ (병렬)
② [탐색] tax_knowledge: 각 감면 유형별 조건
③ [탐색] cli-law: 감면 근거 법령
    │
    ▼
④ [응답] 해당 가능한 감면 유형 + 조건 + 법적 근거
```

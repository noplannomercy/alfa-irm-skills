## 체이닝 패턴 (복합 질의)

- **"이 민원에 답변 초안 만들어줘"** (질의 #11)
  = 접근(#4b: 새올에서 민원 상세 조회)
  → 접근(#5: 세외수입에서 납부 현황 — 비식별화)
  → 탐색(#1: FAQ에서 관련 답변 검색)
  → 탐색(#2: 매뉴얼에서 절차/기준 검색)
  → 탐색(#3: 법제처 API에서 관련 법령 — 해당 시)
  → 탐색(GraphRAG: 용어 확장 — "세금 깎아달라" → 감면 유형들)
  → 실행(유연: 종합하여 답변 초안 생성)

- **"세금 깎아달라는데 어떤 감면이 해당돼?"** (질의 #9)
  = 탐색(GraphRAG: "감면" → 장애인/다자녀/국가유공자 등 하위 유형 확장)
  → 탐색(#2: 각 감면 유형별 조건 매뉴얼 검색)
  → 탐색(#3: 법제처에서 감면 근거 법령)

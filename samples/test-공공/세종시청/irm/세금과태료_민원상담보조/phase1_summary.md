## 아키텍처 개요

### 행위 유형별 배치

### GraphRAG: 승인 (3/4)
### 설계 신뢰도: ✅ 100%

### 인프라 요약
- LLM: CSAP 인증 클라우드 LLM 또는 로컬 (G-클라우드)
- 임베딩: BGE-m3-ko (로컬) 또는 CSAP 클라우드 임베딩
- 벡터DB: pgvector (HNSW)
- 그래프DB: Apache AGE (같은 PostgreSQL)
- CLI: 3개 (saeol, tax, law)
- 비식별화: 필수 파이프라인

## 예상 일정

| 단계 | 기간 | 내용 |
|------|------|------|
| 선행 절차 | 2~3주 | 간이 PIA + 수의계약 (병렬) |
| 인프라 | 1주 | PostgreSQL(pgvector+AGE) + LLM 환경 |
| 탐색(RAG) | 2주 | FAQ 150건 + 매뉴얼 30건(HWP→Forge) + 법령 + 이력(비식별화) |
| GraphRAG | 2.5주 | 세목/감면/과태료 엔티티 추출 + 관계 구축 |
| 접근(CLI) | 1주 | cli-saeol + cli-tax + cli-law + 비식별화 |
| 실행(유연) | 1주 | 답변 생성 프롬프트 + 근거 표시 + Human-in-the-loop UI |
| 통합 테스트 | 1주 | 체이닝 + 비식별화 검증 + 감사로그 |
| 파일럿 | 2주 | 상담사 5명 실 운영 |
| **합계** | **12.5주** (선행 포함) | 9~10월 효과 보고 가능 |

### 타임라인 (12월 마감, 9~10월 보고 대비)
```
5월 초  ← 선행 절차 (PIA + 수의계약)
5월 중  ← 인프라 세팅
5월 말  ← RAG 컬렉션 구축 + Forge(HWP)
6월    ← GraphRAG + CLI
7월 초  ← 실행 + 통합 테스트
7월 중  ← 파일럿 운영 (2주)
8월    ← 효과 분석 + 보고서 작성
9월    ← 효과 보고 + 내년 예산 확보
```

- 예산: 4,990만원
- 주요 비용: CSAP 클라우드 또는 G-클라우드 인프라, 외부 인력 12주

## CLI 설계

### cli-saeol (새올 민원 조회)
- 명령어:
  - `saeol case get --id <민원ID>` → 민원 상세 (비식별화)
  - `saeol case search --type <세금유형> --from --to` → 세금 민원 검색
  - `saeol case list --status <상태> --limit` → 최근 민원
- 출력: JSON, **자동 비식별화** (주민번호 마스킹, 이름/전화 제거)
- 감사로그: 필수 (공공기관 의무)

### cli-tax (세외수입 조회)
- 명령어:
  - `tax levy get --case-id <민원ID>` → 관련 부과 내역 (비식별화)
  - `tax payment status --case-id <민원ID>` → 납부/체납 현황 (비식별화)
- 출력: JSON, **자동 비식별화**
- 감사로그: 필수
- 접근 모드: 읽기전용

### cli-law (법제처 API)
- 명령어:
  - `law search --keyword <키워드>` → 관련 법령 검색
  - `law article --law <법령명> --article <조항>` → 특정 조항 조회
- 출력: JSON
- 접근: REST API (공개, 인증 불필요)

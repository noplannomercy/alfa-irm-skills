## 구현 순서

```
Phase 0: 선행 절차 (2~3주, 개발과 병렬)
  └─ 간이 PIA + 수의계약

Phase A: 인프라 (1주)
  └─ PostgreSQL + pgvector + AGE + LLM

Phase B: 탐색 (2주)
  ├─ B-1: tax_knowledge (FAQ + 매뉴얼→Forge + 법령)
  └─ B-2: tax_consultation (새올 → 비식별화 → 인덱싱)

Phase C: 접근 (1주) — Phase B와 병렬
  └─ cli-saeol + cli-tax + cli-law + 비식별화 파이프라인

Phase D: GraphRAG (2.5주)
  └─ 세목/감면/과태료 엔티티+관계 구축
  └─ 의존: Phase B (tax_knowledge)

Phase E: 실행 (1주)
  └─ 답변 생성 프롬프트 + 근거 표시 + UI

Phase F: 통합 + 검증 (1주)
  └─ 체이닝 + 비식별화 전수 검증 + 감사로그

Phase G: 파일럿 (2주)
  └─ 상담사 5명 실 운영
```

**합계: 12.5주 (선행 절차 포함)**

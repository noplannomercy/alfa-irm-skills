# 세종시청 고객 프로필

> 작성일: 2026-04-09
> 최종 수정: 2026-04-18
> 상태: 2차 미팅 완료 — 프로필 완성

## 1. 회사 개요
- 업종: 공공 (세종시청)
- 규모: 시민봉사과 민원상담팀 12명
- 민원: 전화 500건/일 + 방문 200건/일 [1차 미팅 확인]
- 민원 유형: 세금/과태료 30%, 복지 25%, 인허가 20%, 교통 15%, 기타 10% [1차 미팅 확인]

## 2. AI/LLM 현황
- 기존 시도: 없음 [1차 미팅 확인]
- 남은 자산: 없음

## 3. 인프라
- 클라우드: G-클라우드 + 자체 전산실. 외부는 CSAP 인증만 가능. [1차/2차 미팅 확인]
- DB: 새올 (DB 직접 조회, REST API 없음), 세외수입 시스템 (DB 직접 조회) [2차 미팅 확인]
- 정부24: 화면 연계만 [1차 미팅 확인]
- 법제처 API: 공개, 바로 사용 가능 [2차 미팅 확인]

## 4. 규제 환경
- 감독기관: 행안부, 개인정보보호위원회 [1차 미팅 확인]
- 개인정보보호 담당: 박도현 정보보안담당관 [2차 미팅 확인]
- 데이터 분류: 주민번호/소득정보 포함 → PIA 대상 (간이 PIA 가능, 2주) [2차 미팅 확인]
- AI 가이드라인: 행안부 "공공분야 AI 활용 가이드라인" — 최종 판단 사람, 비식별화, 설명 가능성 [2차 미팅 확인]
- 비식별화: 주민번호/소득 마스킹 필수, 이름/전화 제거 [2차 미팅 확인]
- 감사로그: 필수 (공공기관 의무) [2차 미팅 확인]
- 정보화사업 심의: 4,990만으로 조정 → 면제 [2차 미팅 확인]
- 조달: 수의계약 가능 (약 2주) [2차 미팅 확인]

## 5. 원하는 것
- **유즈케이스: 세금/과태료 민원 상담 보조 (확정)** [2차 미팅 확인]
  - 범위: 재산세, 자동차세, 주민세, 과태료 관련 민원
  - 내부 도구 (상담사 보조)
  - 복지는 2차 확장
- 데이터 현황:
  - 세무과 FAQ 약 150건 (홈페이지 DB) [2차 미팅 확인]
  - 세무 매뉴얼 약 30건 HWP [2차 미팅 확인]
  - 지방세법 법령 (법제처 API) [2차 미팅 확인]
  - 세금 민원 이력 새올 약 3만건/년 [2차 미팅 확인]
  - 부과/납부 데이터 — 세외수입 시스템 (개인정보 포함) [2차 미팅 확인]

## 6. 제약사항
- 예산: 4,990만원 (심의 면제 조정) [2차 미팅 확인]
- 일정: 12월 말 집행, 9~10월 효과 보고 [1차 미팅 확인]
- 인력: 전산담당 정우성 1명 파트타임 [1차 미팅 확인]
- 조달: 수의계약 2주 [2차 미팅 확인]
- 고객 대면: 내부 상담사만 [1차 미팅 확인]

## 내부 메모
- 거버넌스 무게: Heavy (확정 — 공공 + PIA + 행안부 가이드라인 + 감사로그)
- GraphRAG 신호: 있음 (강한 신호 — 행정 용어 vs 일상 용어 다층 불일치)
  - "건물분 재산세" vs "토지분 재산세" vs "재산세" (세목 계층)
  - "장애인 감면/다자녀 감면/국가유공자 감면" vs "세금 깎아달라" (감면 유형)
  - "주정차 과태료/환경 과태료/건축 과태료" vs "과태료" (과태료 유형)
- 예상 미팅 횟수: 2회 (완료)
- 예상 로드맵 유형: 규제 Heavy 3~4개월 (조달 2주 + 간이 PIA 2주 선행)

# 세종시청 ALFA 입력 프로필 (확정)

> 확정일: 2026-04-18
> 버전: v1.0
> 근거 미팅: 1차(4/11), 2차(4/18)

## 1. 회사 개요
- 업종: 공공 (세종시청 시민봉사과)
- 규모: 민원상담팀 12명, 전화 500건/일 + 방문 200건/일
- 민원 유형: 세금 30%, 복지 25%, 인허가 20%, 교통 15%, 기타 10%

## 2. AI/LLM 현황
- 기존 시도: 없음
- 남은 자산: 없음

## 3. 인프라
- 환경: G-클라우드 + 자체 전산실. CSAP 인증 클라우드만 가능.
- DB: 새올(DB 직접, API 없음), 세외수입(DB 직접)
- 법제처 API: 공개, 사용 가능

## 4. 규제 환경
- 감독: 행안부, 개인정보보호위원회
- PIA: 간이 (2주)
- 가이드라인: 행안부 공공 AI — 최종 판단 사람, 비식별화, 설명 가능성
- 비식별화: 주민번호/소득 마스킹, 이름/전화 제거
- 감사로그: 필수
- 심의: 면제 (4,990만)
- 조달: 수의계약 (2주)

## 5. 원하는 것
- 유즈케이스: 세금/과태료 민원 상담 보조 (내부 도구)
- 범위: 재산세, 자동차세, 주민세, 과태료
- 향후: 복지 → 인허가 → 시민 챗봇

## 6. 제약사항
- 예산: 4,990만원
- 일정: 12월 말, 9~10월 효과 보고
- 인력: 전산담당 1명 파트타임
- 대면: 내부 상담사만
- 선행: PIA 2주 + 수의계약 2주 (병렬 가능)

# 세종시청 1차 미팅 서머리

> 일시: 2026-04-11
> 참석: 김현정 민원상담팀장, 정우성 전산담당, 당사 2명

## 논의 내용

1. **민원 현황**: 민원상담팀 12명, 일 전화 500건 + 방문 200건. 세금/과태료(30%)와 복지(25%)가 전체의 55% 차지.
2. **반복 질문**: 재산세 납부 기한, 기초연금 신청 방법, 영업허가 절차 등이 가장 빈번.
3. **데이터 현황**: FAQ 약 800건(홈페이지 DB), 응대 매뉴얼 약 120건(부서별 문서), 상담 이력 3년치(새올).
4. **인프라**: G-클라우드 + 자체 전산실. 새올은 DB 직접 조회만 가능. 외부 클라우드는 보안인증된 것만.
5. **방향 합의**: 우선 상담사용 내부 보조 도구부터. 세금/복지 분야를 우선 검토.

## 합의 사항
- 내부 도구(상담사 보조) 우선, 시민 직접 챗봇은 향후 검토
- 세금/복지 분야를 파일럿 후보로 우선 검토
- 2차 미팅에 정보보안담당관 동석하여 개인정보/보안 사항 확인

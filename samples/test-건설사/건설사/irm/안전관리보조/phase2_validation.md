| 전체 현장 기상 조회 | 2분 이내 | 8개 현장 병렬 |

### 3-2. 측정 방법

```python
import time

def measure_response_time(query, site_id=None):
    start = time.time()
    response = client.post("/query", json={"query": query, "site_id": site_id, "user_id": "test"})
    elapsed = (time.time() - start) * 1000
    return response.json(), elapsed

# 20개 TC 전체 측정 후 평균/최대값 기록
```

## 6. 사용자 검증 (UAT)

### 6-1. UAT 계획

- 대상: 안전관리자 10명 (현장별 1~2명)
- 기간: 2주 (PoC 전 4주)
- 방식: 실제 업무에 AI 도구 병행 사용

### 6-2. 측정 항목

| 항목 | 측정 방법 | 목표 |
|------|---------|------|
| 정보 탐색 시간 | 스톱워치 측정 (UAT 전후) | 20% 이상 단축 |
| 만족도 | 5점 척도 설문 | 4점 이상 |
| 재사용 의향 | 설문 | "매우 그렇다" 70% 이상 |
| 불편 사항 | 자유 기술 | 주요 이슈 Top 3 도출 |

### 6-3. 설문 항목

```
1. 원하는 정보를 빠르게 찾을 수 있었습니까? (1~5점)
2. 제공된 정보의 정확성에 만족하십니까? (1~5점)
3. 출처 정보가 충분히 제공되었습니까? (1~5점)
4. 기상 판단 보조 기능이 유용하였습니까? (1~5점)
5. TBM 자료 자동 생성 기능이 유용하였습니까? (1~5점)
6. 업무에 지속적으로 사용하고 싶습니까? (1~5점)
7. 개선이 필요한 점을 자유롭게 작성해주세요:
```

## 데이터 품질 리스크 상세 — R-06 (제외 리소스)

| 항목 | 내용 |
|------|------|
| 문제 | OCR 미적용: 스캔 이미지에서 텍스트 추출 불가 |
| 원인 | 구형 스캐너 사용, 도면 스캔 후 OCR 처리 없이 보관 |
| 심각도 | 높음 — 텍스트 추출 실패 시 임베딩 자체가 불가 |
| 품질 필터 기준 (2차 PoC용) | OCR 후 추출 텍스트 500자 이상, Tesseract 신뢰도 70% 이상 |
| OCR 도구 후보 | AWS Textract (도면 외부 전송 허용 시) / Tesseract v5 (로컬 처리) |
| 재검토 조건 | 보안팀 외부 전송 허용 확인 + OCR 처리 후 품질 검증 결과 |

## 1. 도입 배경

### 1-1. 경영 지시 배경

2025년 중대재해처벌법 감사 강화 이후, 안전관리본부장이 다음 두 가지 문제를 경영진에 보고:

1. **인력 효율 문제**: 안전관리자 25명이 안전 정보를 찾는 데 업무 시간의 30%를 소모. 현장별 2~3명이 전국 8개 현장에 분산되어 있어 즉각적인 전문 지원이 어려움.
2. **기준 일관성 문제**: 동일한 위험 상황에 대해 현장마다 다른 안전 조치를 취함. 이는 감사 시 "일관성 없는 안전관리"로 지적될 수 있어 법적 리스크가 됨.

결과: 안전관리본부장 지시 — "AI로 안전관리 효율화, 사고 예방 강화"

### 1-2. 산업 환경 변화

- **중대재해처벌법(2022 시행)**: 중대재해 발생 시 경영책임자 처벌. 안전관리 기록의 완전성이 법적 방어의 핵심
- **산업안전보건법 개정 빈도 증가**: 연 2~3회 이상 시행령/고시 개정, 내부 반영 지연이 법 위반으로 직결
- **건설 현장 사망사고 통계**: 건설업 산업재해 사망자 수 전 산업 1위 (2024 기준), 규제 압박 지속

## 2. 현재 상태 진단

### 2-1. 정보 접근 비효율

| 항목 | 현재 상태 | 문제 |
|------|---------|------|
| 매뉴얼 탐색 | PDF 직접 열기 + Ctrl+F | 연관 정보 누락, 시간 과다 |
| 법규 탐색 | PDF 500p 수동 검색 | 개정본 반영 지연 2~3개월 |
| 사고 사례 | 엑셀 Ctrl+F | 유사 사례 발굴 어려움 |
| 동료 문의 | 전화 문의 | 답변 신뢰도 불균일 |

### 2-2. 기준 불일치 구조

```
본사 안전매뉴얼 (최신본)
    ↓ 갱신 지연 2~3개월
현장별 프린트 배포본
    ↓ 현장별 해석 차이
현장 A: "고소작업" 기준 적용
현장 B: "높이작업" 기준 적용
        → 동일 위험, 다른 안전 조치
```

### 2-3. 법적 리스크 구조

- 중대재해처벌법 제4조: 안전보건관리체계 구축 의무
- 산업안전보건법 제36조: 위험성 평가 실시 의무
- 감사 포인트: 기록의 완전성, 기준의 일관성, 법규 최신 반영 여부

## 3. AI 도입으로 달성하려는 목표

### 3-1. 핵심 목표

| 목표 | 현재 | 목표 (PoC) |
|------|------|-----------|
| 정보 탐색 시간 | 30% | 10% 이하 (20% 단축) |
| 기준 일관성 | 현장마다 다름 | 동일 질의 = 동일 답변 |
| 법규 반영 지연 | 2~3개월 | 당일 반영 (자동화) |
| TBM 사고 사례 활용 | 수동 검색 | 작업 유형별 자동 제시 |

### 3-2. 비재무적 목표

- 안전관리자의 역할이 "정보 검색자"에서 "안전 판단자"로 전환
- 중대재해처벌법 대비 기록 완전성 확보
- 신입 안전관리자 온보딩 기간 단축 (표준 기준 접근 용이)

## 4. AI가 적합한 이유

| 판단 기준 | 대림건설 상황 | AI 적합성 |
|---------|------------|---------|
| 반복적 정보 탐색 | 연 2,400건 점검, 매번 유사한 질의 | ✅ 높음 |
| 비구조적 문서 다수 | PDF 매뉴얼, 법규집, SOP | ✅ RAG 최적 |
| 도메인 전문성 필요 | 안전 법규, 공종별 기준 | ✅ 문서 기반 RAG 적합 |
| 기준 일관성 요구 | 현장 간 동일 기준 적용 | ✅ 중앙화된 지식베이스 |
| 기록 완전성 요구 | 감사 대비 모든 근거 보존 | ✅ 출처 추적 가능 RAG |

## 5. 도입하지 않을 경우 위험

| 위험 | 영향 |
|------|------|
| 법규 미반영 상태 지속 | 산업안전보건법 위반 → 과태료, 영업정지 위험 |
| 중대재해 발생 | 경영책임자 처벌, 기업 이미지 타격 |
| 안전관리자 역량 편차 | 현장별 안전 수준 불균일 → 사고 위험 증가 |
| 경쟁사 대비 지연 | 건설업계 안전 AI 도입 가속화 추세 |

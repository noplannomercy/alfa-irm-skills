## 4. 격차 분석 (Gap Analysis)

### 4-1. 현재 vs 목표

| 영역 | 현재 상태 | 목표 상태 | 격차 |
|------|---------|---------|------|
| 정보 탐색 | PDF 수동 검색, 30% 시간 | 질의 응답, 5초 이내 | RAG 시스템 구축 필요 |
| 기준 일관성 | 현장별 다름 | 중앙화된 동일 답변 | 단일 지식베이스 필요 |
| 법규 반영 | 2~3개월 지연 | 당일 자동 반영 | 갱신 파이프라인 필요 |
| 사례 활용 | 엑셀 수동 | TBM 자동 제시 | 검색 인터페이스 필요 |
| 기상 판단 | 수동 | 자동 권고 | API 연동 + 규칙 엔진 필요 |
| 도면 검색 | 불가 | 안전 표기 검색 | OCR + 품질 필터 필요 (2차) |

### 4-2. 우선 해결 과제

1. **핵심**: 텍스트 데이터(매뉴얼, 법규, SOP, 사고 사례) RAG 구축
2. **중요**: GraphRAG로 동의어/관계 처리 ("고소작업=높이작업")
3. **부가**: 기상 API 연동 + 작업 중지 판단 로직
4. **조건부**: 도면 OCR 전처리 후 품질 기준 충족 시 포함

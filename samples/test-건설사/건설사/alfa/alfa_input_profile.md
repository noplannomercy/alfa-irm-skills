## 확정 기술 스택

| 항목 | 결정 |
|------|------|
| 벡터DB | pgvector (PostgreSQL 확장, 기존 RDS 활용) |
| 그래프DB | Apache AGE (PostgreSQL 확장, 동일 인스턴스) |
| 임베딩 모델 | BGE-m3-ko (한국어 특화, 로컬 처리) |
| LLM | GPT-4o (API, 텍스트 응답) / GPT-4o-mini (도구 선택, 트리플 추출) |
| 파싱 도구 | Docling + HybridChunker |
| 프레임워크 | FastAPI (백엔드), 기존 안전관리 시스템 연동 |
| 인프라 | AWS EC2 (애플리케이션), AWS RDS PostgreSQL (DB) |

1. 안전관리자 정보 탐색 시간 20% 이상 단축
2. 동일 질의에 대한 현장 간 일관된 답변 제공
3. TBM 시 관련 사고 사례 자동 제시 (질의 시 5초 이내 응답)
4. 안전관리자 만족도 4점 이상 / 5점 (내부 설문)

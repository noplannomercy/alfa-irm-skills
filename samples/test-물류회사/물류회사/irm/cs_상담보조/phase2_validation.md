## 엔드투엔드 시나리오

| # | 시나리오 | 체이닝 | 합격 기준 |
|---|---------|-------|----------|
| 1 | 배송 조회 | 접근 | 정확한 현황 ≤2초 |
| 2 | 반품 접수 | 탐색→접근→실행(확정) | 전체 체인 ≤5초, 판단 정확 |
| 3 | 클레임 처리 | 접근→탐색→실행(유연) | 유형 판단 + 보상 제안 |
| 4 | 답변 생성 | 접근→탐색→실행(유연) | 초안 품질 상담사 승인 |

## PoC 범위
- 대상: 배송 조회 + FAQ 검색 (문의의 55~60%)
- 기간: 3.5주
- 참여: 상담사 5명
- 성공 기준: Recall@5 ≥85%, 배송 조회 ≥99%, 검색 시간 30%→15%
- 확대 조건: → 전체 20명 + 클레임/반품

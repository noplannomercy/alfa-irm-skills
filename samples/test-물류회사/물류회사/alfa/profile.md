## 1. 회사 개요
- 업종: 물류/배송
- 규모: 중견 (직원 800명, 연매출 3000억)
- 주력 사업: 전국 물류센터 5개 운영
- 회사명: 한진로지스

## 2. AI/LLM 현황
- 기존 시도: 외부 챗봇 업체 도입
- 결과: 3개월 만에 철수 — FAQ만으로 맥락 부족
- 남은 자산: FAQ 287개 (엑셀 수령)

## 3. 인프라
- 클라우드: AWS (EC2 + RDS)
- DB: MySQL (CS 상담 이력 2년 8개월, ~150만건, ERD 확인)
- CS 시스템: 자체개발 웹 기반
- 택배사 API: CJ(REST, API Key, 12종 상태코드), 한진(REST, OAuth2, 9종), 롯데(REST, API Key, 10종)

## 4. 규제 환경
- 감독기관: 없음
- 개인정보: CS 데이터에 포함 (이름/전화번호/주소)
- 비식별화 기준: 이름 마스킹(홍*동), 전화번호 뒷4자리, 주소 시/구까지만
- LLM 투입 시 마스킹 필수, 원본 저장 금지
- AI 조회 이력 로깅 필요

## 5. 원하는 것
- 경영진 지시: 대표이사 "AI로 CS 효율화"
- 현업 니즈: 매뉴얼 검색 시간 30% 소모 → 검색 보조
- 유즈케이스: CS 상담 보조 (상담사용)
- CS 유형: 배송 조회 40%, 반품/교환 25%, 클레임 20%, 기타 15%

## 6. 제약사항
- 예산: 상반기 3천만원
- 일정: 6월까지 성과
- 인력: IT팀 3명
- 고객 대면: 1차 내부, 2차 고객 셀프서비스

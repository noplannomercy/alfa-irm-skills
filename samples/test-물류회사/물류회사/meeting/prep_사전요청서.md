## A. 사전 질의 (담당자용)

아래 항목 중 현재 파악 가능한 내용을 정리해주세요.

### 현황 관련
- 이전에 AI, 챗봇, CS 자동화 관련 시도하신 경험이 있으시면 어떤 걸 했고 결과가 어땠는지 간단히 정리해주세요
- 현재 CS 상담 시스템이 어떤 제품/방식인지 (자체개발? 외부 솔루션?)
- 클라우드 사용 여부 (AWS, Azure 등)
- CS 문의 유형별 비율을 대략 알려주시면 좋겠습니다 (배송 조회, 반품, 클레임 등)

### 목표/기대 관련
- AI 도입으로 기대하시는 구체적인 효과가 있으시면 알려주세요
- 예산/일정 계획이 잡혀있으시면 공유 부탁드립니다
- IT팀에서 이 프로젝트에 투입 가능한 인력 규모

## B. 리소스 조사 요청 (IT팀용)

AI 도입 범위를 구체화하기 위해 아래 정보가 필요합니다.
미팅 전 또는 미팅 후 1주 내 회신 부탁드립니다.

### 문서/파일
- [ ] CS 상담 관련 문서 목록 (FAQ, 매뉴얼, 절차서 등)
- [ ] 각 문서의 저장 위치, 파일 형식, 대략적 건수
- [ ] 샘플 문서 3~5건 (내용 확인용)

### 시스템/DB
- [ ] CS 상담 시스템 정보 (제품명, DB 종류, 접근 방식)
- [ ] 상담 이력 DB 테이블 구조 또는 주요 필드 목록
- [ ] 데이터 규모 (건수, 보관 기간)
- [ ] 배송 추적 시스템 정보 (택배사별 API 연동 현황)
- [ ] 택배사 API 문서 (있으시면)

### 절차/매뉴얼
- [ ] 정해진 처리 절차가 있는 업무 목록 (반품 처리, 클레임 처리 등)
- [ ] 각 절차의 단계 수, 판단 기준 유무

### 보안/접근
- [ ] CS 데이터에 고객 개인정보 포함 여부 (이름, 전화번호, 주소 등)
- [ ] 개인정보 처리 관련 내부 정책
- [ ] 데이터 외부 반출 가능 여부
- [ ] 클라우드/온프레미스 제약 사항

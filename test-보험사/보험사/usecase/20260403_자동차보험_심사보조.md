# 유즈케이스 정의서: 자동차보험 심사 보조

> 고객: 미래에셋손해보험
> 작성일: 2026-04-03
> 상태: 확정 (리소스 전부 수령)

## 개요
- 목적: 심사역이 자동차보험 심사 시 유사 사례 검색, 약관 적용 안내, 심사 의견 초안을 AI로 보조받음
- 사용자: 심사역 15명 (내부)
- 입력: 보험 청약서 정보, 심사 요청 내용
- 출력: 유사 심사 사례 + 적용 약관 조항 + 심사 의견 초안 + 판단 근거

## GraphRAG 필요성
- **필요 가능성 높음**
- 근거:
  - "같은 자동차 사고인데 심사역마다 판단이 달라요" → 심사 기준 해석 차이
  - 약관 조항 간 관계 복잡 (면책, 특약, 할증 조건이 서로 참조)
  - 유사 사례 검색 시 사고 유형-약관 조항-심사 결정 간 관계 추적 필요
- 유사 케이스: 현대캐피탈 시뮬레이션 (금융, "한도" 동음이의어)

## 1. 회사 개요
- 회사명: 한진로지스
- 업종: 물류/배송
- 규모: 중견 (직원 800명, 연매출 3000억)
- 주력 사업: 전국 물류센터 5개 운영, 배송 서비스
- CS 규모: 상담사 20명, 하루 2000건

## 2. AI/LLM 현황
- 기존 시도: 외부 챗봇 업체 도입 → 3개월 만에 철수
- 실패 원인: FAQ만 넣어서 맥락 부족, 상담 이력/절차 참조 불가
- 남은 자산: FAQ 300개, 반품절차서, 클레임 처리 가이드

## 3. 인프라
- AWS (EC2 + RDS)
- MySQL: CS 상담 이력 3년치 (~220만건)
- CS 시스템: 자체개발 웹 기반
- 택배사 API: CJ대한통운, 한진, 롯데 (각각 별도 API, 상태 코드 체계 다름)

## 4. 규제 환경
- 감독기관: 없음
- CS 데이터에 개인정보 포함 (이름/전화번호/주소)
- 비식별화: 현재 미적용 → LLM 투입 전 마스킹 필요
- 개인정보보호법 준수 필요 (비규제 업종이나 개인정보는 있음)

## 5. 원하는 것
- 대표이사: "AI로 CS 효율화"
- 상담사 매뉴얼 검색 시간 30% → 10% 이하
- 1차: CS 상담 보조 (상담사용)
- 2차: 고객 셀프서비스 챗봇 (향후)
- CS 유형: 배송 조회 40%, 반품/교환 25%, 클레임 20%, 기타 15%

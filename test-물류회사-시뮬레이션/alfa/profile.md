# 물류회사 고객 프로필

> 작성일: 2026-03-26
> 상태: 1차 미팅 완료 — 대부분 확인됨

## 1. 회사 개요
- 업종: 물류/배송
- 규모: 중견 (직원 800명, 연매출 3000억)
- 주력 사업: 전국 물류센터 5개 운영
- 회사명: 한진로지스

## 2. AI/LLM 현황
- 기존 시도: 외부 챗봇 업체 도입 (1차 미팅 확인)
- 결과: 3개월 만에 철수 — "FAQ만 달랑 넣으니까 맥락을 못 잡아서"
- 남은 자산: FAQ 300개 + 반품절차서 + 클레임 처리 가이드 (매뉴얼류)

## 3. 인프라
- 클라우드/온프레미스: AWS (EC2 + RDS)
- DB: MySQL (CS 상담 이력 3년치)
- CS 시스템: 자체개발 웹 기반
- 택배사 연동: CJ대한통운, 한진, 롯데 (각각 API 다름)

## 4. 규제 환경
- 감독기관: 없음 (물류 업종)
- 컴플라이언스팀: 별도 없음
- 데이터 분류: CS 데이터에 고객 개인정보 포함 (이름/전화번호/주소), 접근제어 있으나 비식별화 미적용

## 5. 원하는 것
- 경영진 지시: 대표이사 "AI로 CS 효율화하라", 빠른 성과 원함
- 현업 니즈: 상담사가 매뉴얼 찾는 데 시간의 30% 소모 → 검색 보조
- 유즈케이스: CS 상담 보조 (상담사용, 고객 직접 노출 아님)
- CS 문의 유형: 배송 조회 40%, 반품/교환 25%, 클레임 20%, 기타 15%

## 6. 제약사항
- 예산: 상반기 3천만원
- 일정: 6월까지 성과 필요
- 인력: IT팀 3명 (백엔드 2명 + 인프라 1명)
- 고객 대면 여부: 1차는 내부 상담사 보조, 2차에 고객 셀프서비스 챗봇 희망

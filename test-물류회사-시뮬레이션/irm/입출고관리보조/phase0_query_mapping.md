## 질의별 행위 유형 분포

| 행위 유형 | 질의 수 | 비율 | 대표 질의 |
|----------|--------|------|----------|
| 탐색(내부) | 5/13 | 38% | "이전에 비슷한 파손 건 어떻게 처리했어?" |
| 접근 | 3/13 | 23% | "오늘 A센터 입고 현황 보여줘" |

## 체이닝 패턴 (복합 질의)

- **"이상 건 처리해줘"**
  = 접근(WMS 데이터 확인) → 탐색(유사 이력 검색 + 절차서) → 실행(유연, 처리 방법 판단) → 실행(확정, 기록)

- **"이 입고 건 검수해줘"**
  = 접근(WMS 입고 데이터) → 탐색(검수 기준표) → 실행(확정, 합/불 판단) → 실행(확정, 결과 기록)

- **"파손율 높은 품목 원인 분석해줘"**
  = 접근(통계 데이터) → 탐색(이상 이력, 유사 패턴) → 실행(유연, LLM 원인 분석)

## Phase 0 결론

### 행위 유형 분포 요약
- **주요 유형**: 탐색 38% + 실행 38% = 76% — **탐색+실행 혼합형**
- CS(탐색33%+접근33%)와 완전히 다른 패턴
- 제조기업 불량분석(탐색61%+실행15%)과 유사하되 실행 비중이 더 높음

### GraphRAG 사전 신호
- 동음이의어/용어 불일치: **있음 (강)** — "파손"="훼손"="깨짐"="damage"="broken", 5개 센터 자유 텍스트
- 인과관계 질의 비율: 2/13 (15%) — "파손율 높은 품목 원인", "비슷한 건 어떻게 처리"
- 동일 엔티티 다중 맥락: **있음** — 같은 이상 유형이 센터별/담당자별/시기별로 다르게 기록
- → 사전 판단: **후보 (강)** — 3/4 해당, 제조기업 불량분석과 동일 패턴

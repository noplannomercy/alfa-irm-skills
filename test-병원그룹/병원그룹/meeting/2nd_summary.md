# 병원그룹 2차 미팅 서머리

> 일시: 2026-03-25
> 참석: 의료정보보안팀장, IRB 담당자, 영상의학과 과장

## 논의 내용

1. **병원 규모 및 시스템 현황 보완**
   - 본원 500병상, 분원 각 150~200병상
   - 연간 영상 촬영 약 12만건 (본원 기준)
   - PACS는 Infinitt 제품, 본원/분원 동일, HL7 FHIR 연동 가능
   - IT 투입 가능 인력: 인프라 1명 + 개발 2명 (총 3명)

2. **이전 AI 시도 경험 공유**
   - 작년 외부 업체와 챗봇 파일럿 진행 → 의학용어 이해도가 낮아 중단
   - 이번에는 의료 도메인에 특화된 접근이 필요하다는 공감대 형성

3. **데이터 보안 및 규제 기준 확정**
   - 개인건강정보는 가명처리(이름/생년월일/주민번호 마스킹) 후 활용 가능
   - 가명처리 기준은 보건복지부 가이드라인 준수
   - IRB 간편심의로 진행 가능 (후향적 데이터, 약 2주 소요)
   - AI 결과를 의사가 최종 확인하는 구조 → 내부 보조 도구로 운영
   - 접근권한은 기존 EMR 체계 활용, PACS는 읽기전용 연동
   - AI 조회 이력은 기존 EMR 감사로그에 추가
   - 학습용 데이터는 별도 격리 스토리지에 보관

4. **파일럿 범위 합의**
   - 본원 데이터만으로 먼저 진행 (분원은 1차 제외)
   - CT 판독문부터 시작, 성과 확인 후 MRI 확장 검토
   - 의사결정은 기존 의료정보위원회에서 심의

## 합의 사항
- 예산: 파일럿 기준 5천만원 이내
- 일정: 4월 중순 시작 목표 (IRB 2주 + 데이터 준비 2주 감안)
- 범위: 본원 CT 판독문 보조
- 역할 분담: 실행=의료정보팀, 검토=영상의학과, 승인=의료정보위원회, 보고=CTO
- GPU 서버: 연구팀과 주중 업무시간 우선 사용 합의

- 간호기록 요약은 2순위로 파일럿 성과 확인 후 검토

## 논의 내용

1. **현재 시스템 현황 공유**
   - 자체개발 EMR을 온프레미스 환경에서 운영 중
   - 연구용 GPU 서버(A100 2장) 보유

2. **AI 도입 관심 분야 확인**
   - 1순위: 영상의학과 판독 보조
   - 2순위: 간호기록 요약
   - 의사/간호사가 내부적으로 사용하는 보조 도구 형태

3. **현업 과제 공유**
   - 같은 소견이라도 의사마다 표현이 달라 기존 검색이 잘 안 되는 문제

4. **데이터 보안 원칙 확인**
   - 환자 데이터의 외부 반출 불가 — 모든 처리는 원내 서버에서 수행
   - 개인건강정보 관련 IRB 심의 필요 가능성 인지

## 합의 사항
- AI 도입은 내부 직원(의사/간호사) 보조 도구로 시작
- 클라우드가 아닌 원내 서버 환경에서 진행
- CTO가 프로젝트 스폰서로 추진
- 예산은 올해 안에 파일럿 가능한 수준에서 검토

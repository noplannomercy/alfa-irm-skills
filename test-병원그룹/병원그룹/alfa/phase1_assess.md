# 병원그룹 — Phase 1: ASSESS

> 작성일: 2026-03-25
> 상태: 확정

## 데이터 현황
- 종류: CT 판독문 (텍스트), 영상 메타데이터, EMR 진료 기록
- 규모: 연간 약 12만건 (본원 기준) × 수년치 축적
- 위치: 원내 서버 (PACS + EMR)
- 접근성: 가명처리 후 접근 가능 — IRB 간편심의 승인 후 (2주 예상)

## 인프라 현황
- 클라우드/온프레미스: **온프레미스 전용** — 데이터 외부 반출 절대 불가
- GPU: A100 2장 (연구팀 공유, 주중 업무시간 우선 사용 합의)
- EMR: 자체개발, 온프레미스 서버 운영
- PACS: Infinitt, 본원/분원 동일, HL7 FHIR 연동 가능
- 네트워크: 본원 PoC 범위 — 분원 연동은 미해당 (향후)
- 접근권한: 기존 EMR 권한체계 + PACS 읽기전용 연동
- 감사로그: EMR 감사로그에 AI 조회 이력 추가
- 데이터 보관: 원내 서버, 학습용 데이터셋은 별도 격리 스토리지

## 규제 초스캔
- 업종 규제: 보건복지부/식약처 — 현재 내부 보조수단으로 식약처 인허가 불필요
- 데이터 분류: 개인건강정보(PHI) — 가명처리(이름/생년월일/주민번호 마스킹) 후 활용
- 거버넌스 무게 예상: **Medium** — 내부 전용 + 보조수단 판정 + 가명처리 확립

## 인재/문화
- 내부 IT 인력: 의료정보팀 3명 (인프라 1명 + 개발 2명)
- AI 경험: 있음 — 작년 챗봇 PoC 경험 (실패했으나 프로세스 경험 보유)
- 외부 파트너 필요: 예 — 의료 도메인 특화 AI 구축 지원

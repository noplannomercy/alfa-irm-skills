# CT 판독문 보조 — 검증 기준

> 작성일: 2026-03-25

## 행위 유형별 검증

### 탐색 검증
| 항목 | 기준 | 측정 방법 |
|------|------|----------|
| 검색 정확도 (Recall@5) | ≥80% | 영상의학과 과장 선정 테스트 질의 50개, 관련 판독문이 상위 5건에 포함되는 비율 |
| 검색 정밀도 (Precision@5) | ≥70% | 상위 5건 중 실제 관련 판독문 비율, 전문가 판정 |
| 동의어 검색 개선율 | RAG 대비 +20% 이상 | GraphRAG 질의 확장 ON/OFF 비교 (같은 50건 테스트) |
| 응답 시간 | ≤3초 | 검색 요청~결과 반환 (리랭킹 포함), P95 기준 |
| 컬렉션 커버리지 | 최근 3년 100% | 인덱싱된 판독문 수 vs PACS 원본 수 비교 |

### 접근 검증
| 항목 | 기준 | 측정 방법 |
|------|------|----------|
| PACS CLI 정상 동작 | 모든 명령어 성공률 100% | 명령어당 정상/오류 테스트 각 5건 = 총 30건 |
| EMR CLI 정상 동작 | 모든 명령어 성공률 100% | 명령어당 정상/오류 테스트 각 5건 = 총 30건 |
| REST 응답 시간 | ≤1초 | API 호출 P95, 부하 테스트 (동시 10 요청) |
| 데이터 정합성 | 원본과 100% 일치 | CLI 출력 vs PACS/EMR 직접 조회 교차 검증 10건 |
| 가명처리 적용 | PHI 노출 0건 | `emr patient` 출력에서 실명/주민번호 탐지 검사 100건 |

### 실행 검증
| 항목 | 기준 | 측정 방법 |
|------|------|----------|
| 가명처리 탐지율 | ≥99% | 알려진 PHI 포함 샘플 100건, 미탐지 1건 이하 |
| 가명처리 정밀도 | ≥95% | 오탐지(PHI 아닌데 마스킹) 5건 이하/100건 |
| 감사로그 기록률 | 100% | AI 조회 100건 수행 후 로그 누락 0건 확인 |
| 판독문 초안 수용률 | ≥60% (PoC 목표) | 의사가 수정 없이 또는 최소 수정으로 승인한 비율 |
| 용어 표준화 수락률 | ≥70% | 제안된 용어 치환 중 의사가 수락한 비율 |
| 소견 요약 정확도 | ≥80% | 핵심 소견 누락 없이 요약된 비율, 전문가 판정 |

## 엔드투엔드 검증

### 체이닝 시나리오
| # | 시나리오 | 체이닝 | 기대 결과 | 합격 기준 |
|---|---------|-------|----------|----------|
| 1 | "간 종괴 관련 유사 판독문 찾아서 초안 만들어줘" | 탐색→실행(유연) | "hepatic mass", "liver mass" 등 동의어로 확장 검색 → 유사 사례 기반 초안 생성 | 동의어 확장 동작 + 초안 생성 완료 ≤10초 |
| 2 | "환자 A의 이전 CT 보고, 이번 소견과 비교해줘" | 접근→탐색→실행(유연) | PACS에서 이력 조회 → 이전/현재 판독 비교 요약 생성 | 이력 조회 성공 + 비교 요약 정확 |
| 3 | "새 판독문 작성 후 가명처리까지" | 실행(유연)→실행(확정)→실행(확정) | 초안 생성 → 의사 승인 → 가명처리 → 감사로그 | 전체 체인 완료 + 로그 기록 확인 |
| 4 | "HCC 약어 뭔지 알려주고 관련 판독 사례 보여줘" | 탐색(외부)→탐색(내부) | GraphRAG에서 "HCC=간세포암종" 확인 → 관련 판독문 검색 | 약어 해석 정확 + 관련 사례 반환 |
| 5 | "이 판독문 비표준 용어 표준화해줘" | 탐색(외부)→실행(유연) | GraphRAG로 비표준 식별 → 표준 용어 치환 제안 → 의사 확인 | 제안 생성 완료 + 의사 UI 정상 동작 |

## PoC 범위

- 대상: **본원 CT 판독문** (분원/MRI 제외)
- 데이터: 최근 3년 판독문 (약 36만건)
- 기간: **10주** (인프라 2주 + 개발 6주 + 테스트 2주)
- 참여자: 영상의학과 의사 3~5명 (2주간 파일럿 사용)
- 성공 기준:
  - 검색 정확도 Recall@5 ≥80%
  - 판독문 초안 수용률 ≥60%
  - 응답 시간 P95 ≤3초
  - 의사 만족도 4/5 이상 (파일럿 후 설문)
- 확대 조건: 성공 기준 모두 충족 시 → MRI 판독문 확장 + 분원 데이터 통합 검토
